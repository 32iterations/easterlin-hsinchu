#### 3. 優先級矩陣
**矩陣座標系**:
- X軸：影響廣度（受影響家庭比例 %）
- Y軸：痛點深度（平均強度 1-3分）

**優先級排序**:
1. **高優先級**（右上象限）：
   - 時間類痛點：廣度54.5%，深度3.0，分數4.91

2. **中優先級**（中央象限）：
   - 關係類痛點：廣度36.4%，深度2.5，分數2.28

3. **低優先級**（左下象限）：
   - 健康類痛點：廣度9.1%，深度2.0，分數0.55
   - 資源類痛點：廣度9.1%，深度1.5，分數0.41

**優先級計算公式**:
```
優先級分數 = 出現頻次(%) × 影響人數權重 × 強度權重

範例（時間類）：
- 出現頻次：54.5%
- 影響人數權重：3.0（高影響）
- 強度權重：3.0（高強度）
- 優先級分數：54.5% × 3.0 × 3.0 = 4.91
```

#### 4. 服務缺口分析
**對照基準**: 新竹市現有社福服務（2025年10月）

**長照服務缺口**:
- C級長照站：缺口91%（需196站，現有18站）
- 日照中心：缺口81.5%（需352人容量，現有65人）
- 東區日照缺口：90.8%（最嚴重）

**托育服務缺口**:
- 公共托嬰中心：缺口100%（需3,300-5,280人，現有0個公托中心）
- 準公共托育：缺口40-62%（現有1,988人容量）
- 中籤率：1000:1（超低）

**青少年服務缺口**:
- 青少年活動中心：缺口100%（東區無任何設施）
- 少年福利服務中心：規劃中，尚未定案

**核心洞察**:
> **竹科家庭的痛點不是「缺服務」，而是「服務分散」**

- 東區占新竹市49%人口，但僅24.5%日照容量
- 長輩日照在東區、幼兒托育在北區、青少年課輔在香山
- 光通勤接送就耗盡家長時間與精力

**詳細分析**: 請參閱 `service-gap-report-2025.md`

## 🔑 核心發現與政策建議

### 1. 核心發現
```
竹科家庭的痛點不是「缺服務」，而是「服務分散」
100%的文章涉及工作-家庭平衡問題
54.5%的痛點來自「時間貧窮」（接送、通勤、等待）
0%的痛點來自「金錢貧窮」（推翻「年薪300萬貧戶」標題）
```

### 2. 政策建議
**第一優先**：整合服務，解決時間貧窮
- 將托育+日照+青少年活動中心整合至赤土崎館
- 「一站式」服務減少家長通勤接送時間
- 創造三代共融空間（幼兒、青少年、長輩）

**第二優先**：強化關係支持，緩解社群壓力
- 提供正常化的育兒支持團體
- 建立非競爭性的家長交流平台
- 融入社區營造降低孤立感

**第三優先**：改善健康與資源可近性
- 東區資源分配增加至符合49%人口比例
- 日照中心容量擴充至352人（現有65人）
- 青少年活動中心填補100%缺口

### 3. 赤土崎多功能館定位
**名稱**: 赤土崎全齡社福樞紐（原：多功能館）
**功能**: 托育+日照+青少年「三合一」整合中心
**預算**: NT$230-350M（含規劃設計+工程+營運3年）
**標案**: 114A109，NT$22.87M規劃設計，10/30開標
**時機**: 設計階段（11-12月）為納入整合功能的黃金窗口
