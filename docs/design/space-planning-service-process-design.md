## Step 5: 空間規劃與服務流程設計

- Step 1-4 所有產出檔案
- 新竹縣綜合社福館案例（2.8億、4層樓）
- 赤土崎多功能館標案規格書
- 建築法規（無障礙設施、消防安全）

**任務**：
1. 樓層配置設計：
   - 地下1樓：停車場（家長接送動線）
   - 1樓：日照中心 + 公共托嬰中心（分區隔音）
   - 2樓：親子館 + 課後照顧班
   - 3樓：青少年中心 + 家庭諮商室
   - 4樓：社區共餐 + 長青學苑
   - 每層樓面積分配、動線規劃、設備清單

2. 服務流程設計：
   - 用戶旅程地圖（User Journey Map）：
     * Persona 1: 竹科雙薪家庭（長輩失智+幼兒托育需求）
     * Persona 2: 單親家長（課後照顧+家庭支持需求）
     * Persona 3: 青少年（職涯探索+社群歸屬需求）
   - 繪製「一日生活流程圖」

3. 智慧管理系統：
   - 預約系統流程圖（整合1999專線 + 新竹通APP）
   - 數據儀表板：即時使用率、滿意度、等候人數
   - AI媒合演算法：根據家庭狀況推薦服務組合

**輸出** （請直接進行佈署）：
1. 3D建築模擬圖（使用Three.js或類似工具）
2. 用戶旅程地圖（Mermaid流程圖）
3. 預約系統原型（React互動介面）

**範例：用戶旅程地圖（Mermaid）**：

```mermaid
journey
    title 竹科雙薪家庭的一天（評分：5分=非常滿意 3分=普通）
    section 早上 07:00-09:00

      同時送幼兒至1F托嬰中心: 5: 家長
      前往竹科上班: 3: 家長
    section 下午 17:00-19:00
      下班接小孩至2F課後照顧班: 4: 家長
      參加3F親職講座: 5: 家長
      接長輩回家: 5: 家長
```
