## 📊 执行摘要

### 关键发现

1. **115年度总预算规模创历史新高**: 约460亿元（歳入）、约430亿元（歳出）
2. **统筹分配税款暴增**: 从114年度83亿增至115年度293亿，增加约210-214亿元
3. **社会福利列为五大主轴之一**: 115年度预算主打「零负债、发现金、强社福、稳经济、拚建设」
4. **114年度社福预算占比**: 54.87亿元，占总预算18.43%（第二大支出项目）
5. **财政状况大幅改善**: 预计实现零负债，114年度已达20年来首次盈余

### 2.3 115年度社福预算（推估）

**官方宣示**: 115年度预算五大主轴之一为「**强社福**」

**推估方法一：维持占比法**
```
假设社福占比维持18.43%
115年度社福预算 = 430亿 × 18.43% = 79.25亿元
较114年增加 = 79.25 - 54.87 = 24.38亿元（+44.4%）
```

**推估方法二：增幅比例法**
```
假设社福增幅与总预算增幅相当（+44.8%）
115年度社福预算 = 54.87 × 1.448 = 79.45亿元
较114年增加 = 24.58亿元
```

**推估方法三：财划法效应法**
```
假设新增统筹分配税款的20%用于社福
115年度社福预算 = 54.87 + (210 × 20%) = 96.87亿元
较114年增加 = 42亿元（+76.5%）
```

**保守估计**: 115年度社福预算约 **75-85 亿元**
**乐观估计**: 115年度社福预算约 **85-97 亿元**

## 💡 十一、对提案的建议

### 11.1 预算论述建议

**✅ 可直接引用的数据**:
1. "114年度社会福利预算54.87亿元，占总预算18.43%，居第二位"
2. "财划法修正后，115年度统筹分配税款从83亿增至293亿，增加210亿元"
3. "115年度预算五大主轴之一为'强社福'，显示市府对社会福利的重视"
4. "赤土崎多功能馆已发包标案2,287万元，2025年10月30日开标"

**⚠️ 需标注推估的数据**:
1. "115年度社福预算推估75-97亿元（基于114年占比18.43%推估）"
2. "赤土崎馆工程预算推估1.5-2.3亿元（基于公共建筑造价推估）"

**❌ 不建议直接引用的数据**:
1. 兒少馆、少年中心的具体预算（尚未公开）
2. 长照、托育的细项预算（未公开细项）

### 11.2 时机论述建议

**强调黄金窗口**:
> "赤土崎多功能馆标案10月7日公告，**10月30日即将开标**。开标后将进入规划设计阶段（预计3-6个月），**这是影响功能定位的最后窗口**。若要整合长照日照、公共托嬰、青少年活动等全龄功能，**必须在设计阶段提出需求**。"

**强调财政支持**:
> "财划法修正后，新竹市115年度统筹分配税款增加210亿元，总预算达460亿、社福预算推估75-97亿，**市府有充足财力支持社福建设**。赤土崎全龄社福樞紐的建设成本推估2.8-3.9亿元，**仅占115年度社福预算的3.6-5.2%**，财务可行性极高。"

### 11.3 整合论述建议

**避免推翻市府规划**:
> "新竹市政府规划「兒少家庭福利館」、「少年福利服务中心」、「赤土崎多功能馆」三馆，**显示对全龄社福的重视**。我们不是要推翻这个规划，而是建议**优化整合方式**。"

**提出功能整合方案**:
> "即使三馆分别设置，也应建立**服务流程整合机制**：
> 1. 赤土崎馆设立「全龄社福服务柜台」，整合三馆资讯与预约
> 2. 建立统一的线上预约系统
> 3. 规划联合交通接送服务，减少家庭奔波
> 4. 建议兒少馆、少年中心选址于赤土崎馆步行10分钟范围，形成「东区社福园区」"
