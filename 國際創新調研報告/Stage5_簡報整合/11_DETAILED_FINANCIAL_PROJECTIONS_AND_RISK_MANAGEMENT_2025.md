銀行貸款可貸性評估指標：

1. 現金流覆蓋比 (DSCR)
   【公式】年度淨現金流 / 年度本息和
   第1年：2.25M / (100M/5 + 100M×3.2%) = 2.25M / 23.2M = 9.7% ✗
   第2年：4.1M / 23.2M = 17.7% ✗

   第4年：7.6M / 23.2M = 32.8% ✓
   第5年：9.6M / 23.2M = 41.4% ✓

   評估結論：需在第1-2年期間加強現金流管理，或減少初期貸款額度

2. 債務收入比 (DTI)
   【公式】年度債務服務 / 年度收入
   第1年：23.2M / 36.25M = 63.9% (可接受但偏高)
   第3年：23.2M / 43.4M = 53.4% ✓
   第5年：23.2M / 50.8M = 45.7% ✓✓

   評估結論：中期內（3-5年）債務結構穩健

3. 資產擔保比 (LTV)
   建築物現值估計：NT$200M (根據營造成本 + 土地使用權)
   銀行貸款額度：NT$100M
   LTV = 100M / 200M = 50% ✓✓✓ (銀行標準為60%-80%)

   評估結論：擔保品充分，銀行風險低

   └─ TSMC與MediaTek簽署「投資承諾協議」
