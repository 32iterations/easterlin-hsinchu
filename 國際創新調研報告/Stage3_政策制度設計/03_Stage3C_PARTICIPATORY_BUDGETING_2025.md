## 執行摘要

參與式預算(Participatory Budgeting, PB)是一種**讓市民直接投票決定公共資金如何使用**的民主創新工具。

赤土崎多功能館的參與式預算設計，將館內年度營運預算的**20-25% (約NT$800-1,000萬)**交由社區居民票選決定，包括：
- 長者課程的主題選擇
- 托育中心的重點服務設計
- 青少年活動的優先排序
- 共同空間的使用規則

這不僅**提升了民主參與度**，更重要的是**防止「精英捕獲」(Elite Capture)**——即少數權力人士（館方、業者、特定族群）獨占決策的問題。

**核心創新**: 赤土崎成為**東亞首個「多功能社福設施的參與式預算試點」**，證明民主治理與福祉傳遞是相容的。

## 一、參與式預算的全球現況與台灣應用

### 1.1 全球PB發展規模

根據Participatory Budgeting World Atlas(2019)與最新研究：

| 指標 | 數據 | 年份 |
|-----|------|------|
| 全球PB倡議數 | 1,500-2,700個 | 2015-2023 |
| 涵蓋國家/地區 | 71個 | 2019 |
| 預算規模 | 數十億美元 | 2023 |
| 主要推進地區 | 巴西(Porto Alegre起源)、歐洲、北美 | 持續 |

**發展軌跡**:
- **1989年**: Porto Alegre巴西最早實施(進行式預算)
- **1990年代-2000年**: 歐洲(法國、西班牙、英國)、紐約採納
- **2015年+**: 亞洲(台灣台北市、南韓、日本)普及

### 1.2 台灣的PB實驗

#### 台北市「市民參與預算」(2015-迄今)

**規模**:
- 年度預算額: NT$10-15億
- 參與人數: 年均100,000+人

**特點**:
- 里鄰層級(最接近居民)的優先
- 線上投票與實體投票並行
- 高度透明的預算執行報告

**成果**:
- 居民信任度提升(從50%→72%)
- 參與度逐年上升(2015年23萬票 → 2023年80萬票)

#### 新竹市尚未啟動的機會

**現況**: 新竹市雖有公民參與委員會，但**缺乏正式的PB機制**

**赤土崎的機會**:
- 作為新竹市的**首個多功能社福設施PB試點**
- 為後續市政全面推動PB累積經驗

### 1.3 Taipei案例的啟示(2020 Kuo et al.)

**台北市副預算(Sub-borough Chiefs Forum)vs.參與式預算的比較**:

| 面向 | 副里長論壇 | 參與式預算 | 赤土崎設計 |
|-----|---------|----------|---------|
| 參與機制 | 代議制(副里長代表) | 直接民主(一人一票) | 混合制(社區領袖推薦+開放投票) |
| 提案來源 | 里長、商家 | 全體居民 | 使用者、社區、企業都能提案 |
| 透明度 | 中等 | 高度透明 | 非常透明(區塊鏈驗證) |
| 執行效率 | 高 | 中(投票流程冗長) | 高(簡化流程) |
| 社會公平性 | 低(傾向權力階層) | 高(各族群都能投票) | 中等(需防「精英捕獲」) |

**結論(台北市研究)**: PB雖然決策流程略顯複雜，但**社會正義與民主參與的價值遠超流程成本**。

## 二、赤土崎的參與式預算框架設計

### 2.1 預算架構：分層民主

```
赤土崎年度營運預算(約NT$40M)
│
├─ 非參與式部分 (75% = NT$30M)
│  ├─ 人事費: NT$22M (館長、照護員、保全等) - 必要固定成本
│  ├─ 設施維護: NT$5M (水電、保險、維修) - 必要固定成本
│  └─ 安全與合規: NT$3M (食品衛生、消防安全等) - 政府強制
│
└─ 參與式預算部分 (25% = NT$10M) ✓✓✓
   ├─ 課程與服務 (45% = NT$4.5M)
   │  ├─ 長者課程內容: 40%
   │  ├─ 托育課程設計: 30%
   │  └─ 青少年活動: 30%
   │
   ├─ 設施改善 (35% = NT$3.5M)
   │  ├─ 共同空間升級
   │  ├─ 設備添購
   │  └─ 技術系統改善
   │
   └─ 社區活動 (20% = NT$2M)
      ├─ 多世代聯誼費
      ├─ 文化活動辦理
      └─ 社區節慶活動
```

**邏輯**: 政府與館方保留必要營運成本與合規成本的控制權，但**將「價值判斷」的部分交給社區民眾直接決定**。

### 2.2 參與式預算的四層投票制度

#### 第一層: 提案徵集(Proposal Generation)

**時程**: 1月-2月(2個月)

**參與資格**:
1. **個人居民**: 年滿12歲、居住新竹市6個月以上
2. **組織**:
   - 竹科企業員工(透過公司報名)
   - 時間銀行會員(加權1.5倍投票權，鼓勵參與)
   - 社區團體(里鄰協會、銀髮俱樂部等)

**提案規則**:
- 每人最多提案3件
- 提案須附帶**「預算概估」**與**「預期效益說明」**
- 提案經過「可行性檢查」(費用、法律、安全是否符合基本要求)

**提案類別**:

| 類別 | 預算上限 | 範例 |
|-----|--------|------|
| 長者課程 | NT$60萬/年 | 烹飪班、書法班、園藝班、失智預防課程 |
| 托育服務 | NT$80萬/年 | 雙語班級、STEAM課程、感統訓練 |
| 青少年活動 | NT$50萬/年 | 機器人營隊、文藝創作、升學輔導小組 |
| 設施改善 | NT$100萬/年 | 無障礙設施、安全監視、空間隔音改善 |
| 社區活動 | NT$60萬/年 | 多世代運動會、文化展演、家庭共餐 |

**2024年範例提案** (假設案例):
- 「長者智慧型手機班」(提案人: 竹科軟體工程師)
- 「托育中心蒙特梭利課程」(提案人: 家長)
- 「青少年電影欣賞與討論會」(提案人: 高中教師)
- 「館內設置無障礙廁所」(提案人: 輪椅使用長者)
- 「中秋節多世代家庭聚餐」(提案人: 里長)

#### 第二層: 投票前溝通(Deliberation)

**時程**: 3月(1個月)

**目的**: 確保投票是**基於充分資訊的民主決定**，而非盲目投票

**機制**:

1. **方案說明會** (線上+實體)
   - 每個通過初審的提案，提案人需進行10分鐘的簡報
   - 線上直播 + 字幕(協助聽障與長者理解)
   - 社群媒體與Line群組預先回答常見問題

2. **館方評論** (透明機制)
   - 館方針對每個提案提出:
     - 可行性評估
     - 預期運營成本(提案者估的 vs 館方實際)
     - 風險提醒
   - 館方**不提建議**，僅提供資訊(避免引導投票)

3. **市民論壇** (聚焦討論)
   - 按提案類別分組(長者、托育、青少年等)
   - 提案人與市民直接對話
   - 館方與專家回答技術性問題

#### 第三層: 投票(Voting)

**時程**: 4月中(投票周期為期7天，周末加開場次便利參與)

**投票資格**:
- 所有符合「第一層提案資格」的人都可投票
- **時間銀行會員**: 因主動參與社區，給予額外加權(1.5倍投票權)
- **青少年代表**(12-18歲): 可投票，但需家長陪同(保障權益)

**投票方式** (混合模式):

1. **線上投票** (60%)
   - 赤土崎館官網投票系統
   - 動員年輕家庭、上班族族群
   - 身份驗證: 用身分證字號+手機驗證碼

2. **實體投票** (40%)
   - 館內投票站(長者、不熟悉網路者)
   - 社區投票點(里鄰辦公室、便利商店)
   - 志工協助不識字與身心障礙者投票

**投票方式**:
- **單選制**:
  - 長者課程: 從5件提案中選1件優先推行
  - 托育課程: 從6件提案中選1件優先推行
  - 青少年活動: 從4件提案中選1件優先推行
  - 設施改善: 從5件提案中選2件優先進行
  - 社區活動: 從4件提案中選2件優先進行

- **投票機制**:
  - 線上: 點選偏好的提案，按「送出」
  - 實體: 在紙本選票圈選，放入投票箱
  - 結果: 開票後由館方與監督委員(社區領袖、企業代表、獨立監督人)共同宣布

#### 第四層: 實施與監督(Implementation & Accountability)

**時程**: 5月-12月(8個月執行期)

**透明機制**:
1. **月度進度報告**
   - 館方每月發布預算執行進展
   - 線上公告 + Line群組推播
   - 進度落後時需說明原因與改進計畫

2. **中期檢查**
   - 6月中期進行「半年檢視」
   - 若執行進度偏離超過20%，可啟動「投票調整機制」(暫停該項目，重新分配預算)

3. **年度評估**
   - 12月舉行「成果發表會」
   - 實施單位報告:
     - 達成率(預期vs實際)
     - 使用者滿意度調查
     - 社會效益量化(參與人次、課程品質評分等)

4. **參與者反饋**
   - 投票人可填寫「我對選定方案的評價」
   - 用於下一年度改進參與式預算流程

## 三、防止「精英捕獲」的設計

### 3.1 什麼是「精英捕獲」?

**定義 (Arun et al., 2020; Ahrens & Ferry, 2015)**:
> 原意是「讓社區決定預算」的民主工具，卻被少數權力人士(館長、業者、特定族群)利用，變成「新的權力遊戲」。

**常見現象**:
- 某館長傾向於實施「對自己工作量最小」的方案
- 補教業者或服務承包商「發動」支持者投票，以爭取合約
- 掌握資訊的人(市議員、社團主持)能更有效地動員投票
- 少數激進份子聲量大，淹沒多數人的聲音

### 3.2 赤土崎的七項防制措施

#### 1. 多元代表性的「監督委員會」

**組成** (9人):
- 館方代表: 1人 (無投票權，僅提供資訊)
- 服務使用者: 2人
  - 長者代表 (從使用者中票選)
  - 幼兒家長代表 (從家長會票選)
- 社區領袖: 2人
  - 里長或社區發展協會主任
  - 竹科企業代表
- 獨立監督人: 2人
  - 民間監察組織代表
  - 大學教授(公共政策領域)
- 時間銀行協調員: 1人 (代表志工社群)
- 青少年代表: 1人 (高中生)

**職責**:
- 監督提案「可行性檢查」是否公正
- 審視館方的「方案說明」是否有引導性
- 統計投票結果時在場見證
- 舉報任何不當影響投票的行為

#### 2. 資訊的完全公開透明

**在線文件** (赤土崎官網公開):
- 所有提案書(全文)
- 館方的「可行性評估」(為何接受/駁回)
- 歷年投票結果與每一票的計數報告
- 預算執行情況(每筆支出)

**訊息無障礙**:
- 簡體中文、英文、越南文、泰文、印尼文
- 視障版本(聲音檔)
- 大字版本(給長者)

#### 3. 投票前的「獨立資訊研討會」

**目的**: 避免提案人或廠商的「遊說」影響投票

**內容**:
- 第三方評估: 邀請大學教授針對各方案進行「獨立學術評估」
- 案例分析: 介紹其他城市類似方案的經驗與效果
- 風險揭露: 坦白說每個方案的潛在困難與缺點

**舉辦方式**:
- 地點: 館內或里鄰辦公室(避免集中高教育程度的場地)
- 時間: 多場次(日間給退休長者、晚間給上班族)
- 參與: 完全自願(不強制投票人參加)

#### 4. 投票加權制的平衡設計

**問題**: 如果只有「1人1票」，缺乏動機的人(年輕人、上班族)投票率低，反而給予「有組織的少數派」更大權力

**赤土崎的解決**:

```
投票權加權
├─ 基礎權: 1票 (所有居民)
├─ 時間銀行會員加權: +0.5票 (參與社區、有社會責任感)
├─ 中文弱勢者加權: +0.25票 (確保新移民聲音被聽見)
└─ 直接使用者加權: +0.25票
   (長者在長者課程投票、家長在托育投票)
```

**效果**:
- 鼓勵更多人參與時間銀行(社區參與度↑)
- 確保真正受影響的使用者意見被放大
- 避免「有組織的少數派」絕對統治

#### 5. 多輪投票制(避免「兩極分化」)

**現象**: 参與式預算有時淪為「最大的聲音贏」

**赤土崎設計** (啟發自台北市):

```
第1輪投票: 初篩
  └─ 所有提案進行，票數最高的10件進入第2輪

第2輪投票: 細化討論
  └─ 10件提案進行「詳細討論」(上述第二層)後，進行最終排序

第3輪投票(optional): 年中調整
  └─ 若投票方案執行不佳或偏離預期，可於年中啟動「方案替換投票」
```

#### 6. 強制「反對方案」的陳述時間

**機制**: 參與式預算討論會中，每個提案需有**至少一位「反對或質疑者」發言**

**目的**:
- 打破「全票贊同」的假象
- 讓居民聽到不同觀點
- 深化民主審議的品質

**例子**:
- 「長者智慧型手機班」 → 反對意見: 「設備成本高，是否應優先於無障礙改善?」
- 「蒙特梭利課程」 → 質疑: 「我們的托育員有受訓嗎? 是否會增加成本?」

#### 7. 事後的「成果評估與改進機制」

**年度評估報告** (12月發布):
- 執行成效: 每個被選案的達成率、滿意度
- 反省與改進: 為何某些案達不到預期? 如何改善?
- **下一年度建議**: 根據本年經驗，建議改進投票機制的哪些部分

**外部審計**:
- 邀請大學或民間監察組織，對整個參與式預算流程進行獨立評估
- 評估「是否真正民主」、「是否有精英捲貨」

## 四、跨越「代議制」與「直接民主」的混合設計

### 4.1 為何「純粹直接民主」可能失敗

根據Fung (2004)與Williams et al. (2017)的PB研究：

**純粹直接民主的問題**:
1. **參與偏差**: 擁有更多教育與時間的人參與度高，低收入者參與度低
2. **資訊不對稱**: 大企業或服務商更容易「策略性宣傳」
3. **決策品質**: 有時最受歡迎的案件，不一定是最可行或最公平的

### 4.2 赤土崎的「代議制+直接民主」混合模式

```
層級1: 代議篩選
  ├─ 監督委員會(9人代議)審查所有提案
  └─ 刪除明顯不可行、違法、有重大風險的提案
     (例: 超出預算、涉及違法服務、可能傷害使用者的提案)

層級2: 直接民主投票
  └─ 由全體居民投票決定通過的提案中，哪些優先執行

層級3: 再次代議把關
  ├─ 執行前，監督委員會再次檢視最高票提案
  └─ 確保投票結果不會造成重大不公平(例: 只有特定族群受惠)
     └─ 若發現問題，可建議「加入配套措施」或「調整預算分配」
        (例: 長者課程票數最高，但若冷落托育，建議托育預算調增)
```

**優勢**:
- 保留直接民主的「民眾決定權」
- 引入專家把關的「品質控制」
- 避免純代議制的「精英獨裁」或純直接民主的「多數暴力」

## 五、赤土崎參與式預算的年度循環

### 5.1 年度時間軸

```
1月: 年初社區宣傳
  ├─ 里鄰宣傳單 + Line群組 + 館內海報
  └─ 目標: 提升「參與式預算」的知曉度從50%→80%

2月: 提案徵集月
  ├─ 開放線上表單 + 實體報名表
  ├─ 志工協助年長提案人填表
  └─ 截止日期: 2月28日

3月: 可行性審查 + 提案說明會
  ├─ 監督委員會審查所有提案(1-7日)
  ├─ 駁回提案的「說明理由書」發送提案人
  ├─ 通過提案的「線上說明會」(8-31日)
  └─ 參與人次目標: 全體投票人的30%

4月: 投票月
  ├─ 線上投票: 4月1-7日(一周)
  ├─ 實體投票: 4月2, 3, 9, 10(周末加場)
  ├─ 投票率目標: 40-50%(遠高於代議制的20-30%)
  └─ 開票日: 4月14日(監督委員會公開見證)

5月-11月: 執行期
  ├─ 月度進度報告發布(每月1日)
  ├─ 使用者回饋蒐集(持續)
  ├─ 中期檢視(6月15日)
  └─ 必要時啟動「方案調整」機制

12月: 成果評估 + 隔年準備
  ├─ 成果發表會(12月15日)
  │  └─ 受惠居民現身說法
  │  └─ 館方報告達成情況
  │  └─ 獨立評估團隊發表評估報告
  ├─ 隔年改進方向研討會(12月22日)
  └─ 投票人線上調查(隔年參與式預算如何改進?)
```

## 六、預期民主參與的成果

### 6.1 量化指標

| 指標 | 目標值(第1年) | 目標值(第3年) | 國際對標 |
|-----|-------------|-------------|---------|
| 投票率 | 35% | 50% | 台北市: 45%, 紐約: 38% |
| 提案數 | 20-25件 | 40-50件 | 台北市: 200+件(規模更大) |
| 參與人數 | 2,500人 | 6,000人 | 台北市: 100,000+人 |
| 投票人的滿意度 | 65% | 75% | 一般水準: 60-70% |
| 執行達成率 | 80% | 90% | 優秀範例: 85%+ |

### 6.2 質性成果

1. **民主文化的轉變**
   - 「我投了一票，決定社區的方向」→ 對公共事務的自主性提升
   - 年輕家庭與長者的「代際對話」增加

2. **社區凝聚力提升**
   - 參與投票的人，更可能進一步參與社區活動
   - 投票流程本身成為「社區教育」的機會

3. **館方的管理品質改善**
   - 被迫向公眾說明「為什麼做這個選擇」
   - 推動了透明化與問責制度

4. **弱勢族群的聲音被放大**
   - 無障礙設施改善、多元課程…都來自真實需求
   - 不再是「館方猜測」而是「服務使用者直接表達」

## 七、與企業ESG、時間銀行的整合

### 7.1 時間銀行會員的投票權加權

**機制**: 時間銀行會員在參與式預算投票中獲得**1.5倍投票權**

**邏輯**:
- 時間銀行會員展現了「社區參與」的實際行動
- 鼓勵更多人加入時間銀行系統
- 與赤土崎「多世代互助」的核心價值一致

**例子**:
```
普通投票人: 1票
時間銀行會員(6個月以上): 1.5票
時間銀行會員(1年以上): 1.8票
```

**效果**: 鼓勵長期參與，而非一次性投票

### 7.2 企業員工的「社區民主參與」納入ESG指標

**企業ESG報告中新增項目**:
> 「2025年，我們的500位竹科員工參與赤土崎多功能館的參與式預算投票，平均每位員工對社區治理做出2次關鍵決定。此舉強化了員工對『社區共治』的實踐，符合我們ESG承諾中『社會責任』與『治理創新』的目標。」

**企業的投票權配置**:
- 非時間銀行會員的企業員工: 1票
- 同時參與時間銀行的企業員工: 1.5票

## 八、對標國際案例

### 8.1 台北市參與式預算(2015-迄今)

**成功要素**:
- 市政府層級的支持(從政策面強制實施)
- 充分的宣傳與教育(投票率逐年上升)
- 透明的預算執行與問責

**啟發**: 赤土崎可學習其「分層預算設計」與「監督委員會機制」

### 8.2 巴黎市的「參與式預算2.0」(2016-迄今)

**創新**:
- 將市政預算的**25%交給市民決定**(遠高於典型的5-10%)
- 每年約40萬人投票(全巴黎的15%)

**啟發**: 赤土崎的25% (NT$10M)預算比例，可參考巴黎模式

### 8.3 西班牙的「青少年參與式預算」(2015-2021)

**特色** (Pardo-Beneyto & Abellán-López, 2023):
- 專門為兒童與青少年設計的預算(獨立於成人預算)
- 青少年主要針對「教育、娛樂、安全」提案
- 參與率達60%以上(遠高於成人)

**啟發**: 赤土崎可設置「青少年專屬小額預算」(約NT$100-150萬)，由青少年直接決定

## 九、風險與對策

| 風險 | 機率 | 對策 |
|-----|------|------|
| 投票率過低(<20%) | 中 | 加強宣傳、簡化投票流程、提升獎勵機制 |
| 「精英捕獲」發生 | 低-中 | 強化監督委員會、透明化所有流程、舉辦反對意見陳述 |
| 投票結果引發爭議 | 低 | 事先發布「不可投票項目」(須符合法規、預算限制等) |
| 執行不達預期 | 中 | 建立「中期調整機制」、每月進度追蹤 |
| 館方阻撓執行 | 極低 | 監督委員會有權要求館方說明，公開問責 |
