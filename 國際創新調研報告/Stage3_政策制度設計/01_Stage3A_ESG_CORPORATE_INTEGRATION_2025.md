## 執行摘要

2025年台灣進入強制性ESG報告時代，所有上市公司必須提交完整的環境、社會、治理報告。**赤土崎多功能館作為竹科家庭支持樞紐的重要契機**，在於它能整合新竹科技產業的企業ESG戰略，特別是**「社會」維度的制度化落實**。

本報告提出**「三層ESG整合框架」**，將赤土崎館建設與營運納入企業永續報告，同時創造企業ESG績效與社區福祉的雙重價值。

### 關鍵發現
1. **政策紅利** (2025年強制報告時代啟動)
2. **地域優勢** (新竹為全台ESG績效最好的科技聚落)
3. **企業需求** (TSMC、MediaTek等科技龍頭急需「社會責任」落地)
4. **市民期待** (竹科家庭對「全齡照護」有強烈需求)

## 一、 台灣ESG發展現狀與2025政策轉折

### 1.1 強制報告時代的到來

根據台灣金融監督管理委員會(FSC)的Corporate Governance 3.0路線圖，2025年台灣進入**全面強制ESG報告時代**：

| 報告指標 | 2024年 | 2025年 | 變化 |
|---------|-------|-------|------|
| 強制報告公司家數 | 269家(資本額2億以上) | 1,500+家(資本額100萬以上) | +450% ✓ |
| 報告標準 | GRI、SASB併行 | ISSB國際標準優先 | 統一化 |
| 揭露密度 | 選擇性揭露 | 完整性揭露要求 | 強制性↑ |
| 驗證要求 | 自願驗證(60%) | 強制第三方驗證(80%) | 可信度↑ |

**意義**: 台灣企業不再有「不報導」的選項，而是進入**「報導品質競爭」**的時代。

根據Wiley Scholar Gateway研究(Wu & Nguyen, 2024)，台灣上市公司的ESG實踐對會計績效與市場績效都有顯著正影響：
- ESG調整後的Tobin's Q增加0.240 (p<0.01)
- 調整後的市淨比增加0.662 (p<0.01)

### 1.2 台灣半導體產業的ESG領先地位

新竹作為全球最大的半導體聚落，其ESG績效全球領先：

**台灣在全球ESG報告中的位置** (KPMG, 2022):
- GRI標準報告: 第2位 (99%)
- 股票交易所指南: 第5位 (82%)
- SASB標準: 第10位 (52%)

特別在**社會責任面向(Social)**，新竹竹科企業面臨的挑戰是：**「ESG報告寫得好，但社會實踐落地差」**。

**現況問題分析**:
- TSMC、MediaTek的ESG報告側重「環境」(碳中和、能源管理)與「治理」(董事會結構、薪酬透明)
- 但「社會」維度多限於內部員工福利、供應鏈勞動條件
- **缺乏**: 社區層級的長期社會投資、多世代照護系統設計、本地社區福祉承諾

## 二、 赤土崎館作為企業ESG「社會」維度的制度化落地

### 2.1 三層ESG整合框架

```
第1層: 企業層級 (Corporate Level)
  ├─ TSMC、MediaTek、聯電、日月光等竹科龍頭
  ├─ 將赤土崎館納入企業永續報告「社會責任」項目
  └─ 創造5-10年的長期社會投資承諾

第2層: 地方政策層級 (Municipal Policy Level)
  ├─ 新竹市政府社會處主導的公共-民營夥伴(PPP)模式
  ├─ 將赤土崎館定位為「企業ESG最佳實踐案例」
  └─ 與中央財劃法修正後的資源相結合

第3層: 社區層級 (Community Level)
  ├─ 竹科居民(年薪300萬+)的全齡照護需求滿足
  ├─ 時間銀行、參與式預算的實驗場域
  └─ 社區的「社會資本」量化與價值實現
```

### 2.2 企業ESG承諾的具體形式

#### 金融承諾
- **一次性投資**: 新竹市政府標案114A109 (2,287萬)的設計、監造成本
  - 企業可透過「ESG社會責任基金」贊助設計費的20-30% (約600-700萬)
  - 在企業ESG報告中計為「社區福祉投資」

- **營運期贊助** (5-10年期):
  - 每年贊助赤土崎館運營預算的15-25%
  - 包括: 長照服務補助、托育補助、青少年活動經費
  - 相當於TSMC、MediaTek等龍頭的年度ESG預算的0.5-1%

#### 非金融承諾
1. **員工志願服務** (時間銀行接軌)
   - 竹科員工每年可投入50-100小時的志願服務
   - 包括: 長輩陪同、幼兒教室、青少年輔導
   - 累積的「時間學分」可供員工家庭使用

2. **技術與知識移轉**
   - TSMC環保工程師協助赤土崎館的綠色建築認證
   - MediaTek軟體工程師開發服務管理APP
   - 智慧監測系統(感測器、物聯網)的實驗場域

3. **治理參與**
   - 企業代表進入赤土崎館的董事會/監委會 (1-2席)
   - 確保企業視角(效率、創新、風險管理)的納入
   - 定期ESG績效報告與信息透明度

## 三、 ESG整合對赤土崎館的四層價值創造

### 3.1 環境(E)維度：永續設施認證

**標竿**: 台灣第一個達到淨零碳的多功能社福設施

#### 具體指標
| 面向 | 目標 | 時間框架 | 企業角色 |
|-----|------|--------|---------|
| 碳排放 | 淨零碳(Scope 1,2) | 2030年 | TSMC協助碳查核、能源管理 |
| 再生能源 | 100% (含購電協議) | 2028年 | MediaTek協助綠電配置 |
| 廢棄物回收 | 零廢棄目標 | 2027年 | 竹科製造業供應鏈經驗 |
| 水資源 | 60%節水(相比同類設施) | 2026年 | 晶圓廠用水效率經驗 |

**ESG報告價值**: 企業可在年度永續報告中披露「支持社福設施的減碳貢獻」，這對**減碳承諾的可信度極重要**。

### 3.2 社會(S)維度：社區福祉指標的量化

**標竿**: 創造東亞首個「多世代居住滿意度」的標準化量表

#### 核心指標框架

```
社會價值創造 (Social Value Created)
│
├─ 長者面向 (Elderly)
│  ├─ 生活品質指標: ADL/IADL改善率 (目標: 30%)
│  ├─ 社會連結: 每月參與活動次數 (目標: 8次)
│  ├─ 健康指標: 認知功能維持率 (目標: 80%)
│  └─ 滿意度: NPS分數 (目標: 70分)
│
├─ 幼兒面向 (Young Children)
│  ├─ 發展里程碑達成率 (目標: 95%)
│  ├─ 家庭親子互動時間增加 (目標: +2小時/周)
│  ├─ 教保人員-幼兒比改善 (目標: 1:8達到1:6)
│  └─ 托育成本節省 (年均: NT$36,000/家庭)
│
└─ 青少年面向 (Adolescent)
   ├─ 課業表現: 平均成績進步 (目標: +5%)
   ├─ 社交適應: 社團參與度 (目標: 70%)
   ├─ 心理健康: 焦慮症狀改善 (目標: 40%)
   └─ 升學率: 繁星推薦錄取率 (目標: +15%)
```

**ESG報告價值**: 企業可在永續報告「社會指標」章節中披露對這些社區福祉改善的**具體量化貢獻**，勝於空泛的「社區贊助」敘述。

### 3.3 治理(G)維度：多利益相關人治理機制

**標竿**: 建立東亞首個「社福-企業共治」的機構治理模型

#### 治理結構設計

```
赤土崎館董事會/監事會 (11人制)
│
├─ 社會處代表 (2位) - 政府角色
├─ 館長/執行長 (1位) - 營運統籌
├─ 企業代表 (2-3位)
│  ├─ TSMC/MediaTek/聯電推舉代表
│  ├─ 負責監督ESG承諾的落實
│  └─ 提供技術/管理顧問
├─ 使用者代表 (2位)
│  ├─ 長者代表 (由使用者票選)
│  └─ 家長代表 (由家長會推舉)
├─ 社區領袖 (1位)
│  ├─ 鄰里長或社區發展協會主任
│  └─ 代表社區整體利益
├─ 專業人士 (2位)
│  ├─ 社福專家、財務/法律專業
│  └─ 確保專業管治
└─ 獨立董事 (1位) - 監督整體治理品質
```

#### 企業ESG治理責任

1. **季度報告機制**:
   - 企業董事每季向會議報告ESG承諾的進展
   - 包括: 資金投入、志工小時數、技術貢獻度量

2. **年度審驗機制**:
   - 第三方驗證機構(德勤、安永等)每年審驗
   - 企業的ESG貢獻是否達到承諾的85%以上

3. **十年目標制定**:
   - 2025-2035年的長期ESG目標
   - 包括: 設施的環保認證升級、社會指標改善、治理透明度提升

## 四、 競爭優勢：赤土崎館成為企業ESG最佳實踐案例

### 4.1 對企業的五大價值主張

#### 1. 規避ESG報告「形式合規陷阱」
- **問題**: 許多台灣科技企業的ESG報告在「社會責任」章節相對薄弱
  - 內部員工福利講得很細 (薪酬、健康檢查、訓練預算)
  - 社區層級的投資形成卻較少 (多為慈善捐款，非系統性)
- **赤土崎方案的解決**: 提供**可量化、可驗證、有社區影響**的社會責任承諾

#### 2. 獲得「社會責任領導力」認證
- 赤土崎館成為全球ESG評級機構(MSCI、Bloomberg、Sustainalytics)的**標竿案例**
- 企業在ESG評級中獲得「社區夥伴關係」加分項
- 特別對向國際投資者吸引資金有幫助

#### 3. 釋放員工ESG參與的內在動機
- 竹科員工對「社會貢獻」的需求強烈 (受教育程度高、社會責任意識強)
- 透過時間銀行、志願服務，員工能**直接體驗ESG的社區影響**
- 提高員工敬業度、留任率

#### 4. 解決竹科「人才吸引」難題
- 竹科人才外流的重要原因: **「高薪但生活品質不足」**
- 赤土崎館完善的全齡照護，直接解決科技人才的「家庭困境」
- 企業投資赤土崎 = 投資自身人才的生活品質 = 提高人才競爭力

#### 5. 對標全球企業的社會責任標竿
- Apple、Google、Microsoft等科技巨頭在全球推動「社區基礎設施投資」
- 台灣TSMC、MediaTek投資赤土崎，等於向國際展示**本地社區承諾**

**企業年度承諾**:
- TSMC: NT$5-7M/年
- MediaTek: NT$3-5M/年
- 聯電+日月光+其他: NT$2-3M/年
- **合計企業贊助**: NT$10-15M/年 (占營運預算的25-35%)

## 八、 對標案例與國際經驗

### 8.1 新加坡的企業社區投資模型

新加坡政府與企業合作，建立多個「企業贊助的社區中心」：

- **Tampines Family Service Centre** (2015-): 由HP、Google等科技企業贊助
- **績效**: 服務3,000+家庭，企業ESG評級大幅提升
- **經驗**: 企業願投資2-3年的初期虧損，以換取長期社會影響與品牌價值

### 8.2 日本的「時間銀行×企業ESG」整合

Takao et al. (2021)的日本研究顯示，將時間銀行與企業員工志願服務整合時：
- 員工參與度增加40-60%
- 企業被員工認為「有社會責任」的比例從45%升至82%
- ESG評級提升0.5分(在100分制下)
