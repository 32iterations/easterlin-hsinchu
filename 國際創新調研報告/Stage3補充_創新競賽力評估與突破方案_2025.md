## 第一部分 | 當前方案的「創新盤點」

### 1.1 已有全球市場的技術

#### A. 嬰兒監測穿戴設備

**現狀**:
- Owlet Smart Sock: 97.3%靈敏度(FDA認證), 價格USD$299/個
- Snuza Hero: 98%靈敏度, 價格USD$200/個
- 賣點: 即時心率+血氧推播

**赤土崎館的差異**:
- 我們也採購「98%靈敏度」產品 ✓
- 但沒有超越現有產品的獨特性 ⚠️
- **問題**: 評審看到這些設備→想「為何不直接買現成產品」

#### B. AI互動投影

**全球現有方案**:
- 三星Flip & Hub: AI簡報輔助(2023年推出)
- Google Hub: 行事曆與視訊(2021年推出)
- Sony投影機AI音聲辨識(2024年推出)

**赤土崎館的差異**:
- 我們的「50個投影點」規模更大 ✓
- 跨代應用場景是台灣首創 ✓
- 但核心AI模組(姿勢識別、情感分析)使用現有開源庫(MediaPipe, OpenPose) ⚠️

#### C. 時間銀行自動化

**全球現有方案**:
- 新竹市已試辦時間銀行(2023-2025)
- 台北市「銀髮圓夢」時間銀行(結合志工)
- 國際: 日本LETS系統、英國Time Credits

**赤土崎館的差異**:
- NLP配對演算法提升效率96% ✓
- 整合進社福設施(不只是社區銀行) ✓
- 但演算法原理(cosine similarity, vector embedding)是標準ML技術 ⚠️

**啟示**:

#### 趨勢2: 社會影響力債券在老年照護應用加速

**數據**:
- 全球社會金融市場規模: USD 23.3B (2024)
- 但「老年照護SIB」案例<10個(相比健康教育SIB>50個)
- 機會點: **台灣若推出台灣首個「老年+幼兒整合SIB」，將引起國際關注**

#### 趨勢3: 生成式AI在長照應用剛起步

**全球案例**:
- Mayo Clinic (美國): AI大模型用於診斷輔助(2023年)
- Cogito (以色列): AI情感分析於長照人員培訓(2024年)
- 但「生成式AI個性化長者陪伴」的落地案例仍<5個

**機會點**: **赤土崎館若能落實「AI陪伴阿公虛擬化身」，將領先全球3-5年**

### 3.2 突破方案B: 預測性健康分析系統 (+0.35分)

#### 核心創新
```
現狀: 被動監測(發生時才知道)
突破: 主動預警(未來7天誰會有風險)

技術原理:
假設:
├─ 長者A: 過去3天走路速度↓10%, 心率變異度↑20%, 進食量↓
├─ 長者B: 過去1週睡眠時間↓2小時, 停留在安靜室時間↑30分鐘
└─ → 機器學習模型預測: 「下週長者A有50%風險肺炎, 長者B有60%憂鬱症風險」

資料來源 (50個感測器):
├─ 生理: 心率、血氧、體溫、呼吸率
├─ 活動: 走路速度、停留時間、活動次數
├─ 社交: 進食時間、與人互動頻率
├─ 認知: 遠期記憶測試、注意力測試
└─ 環境: 溫度、濕度、氣壓(可能影響老人行為)

預測模型:
├─ 演算法: XGBoost + 時間序列分析
├─ 訓練資料: 全台灣50個長照機構(5,000+長者, 2年數據)
├─ 準確度目標: 72%(>隨機猜測50%)
└─ 預測時間: 7天內發生的不良事件
```

#### 應用場景
```
場景1: 自動預警系統
時間: 每日 08:00 護理主任收到報告
內容:
├─ 綠色: 安全 (50人)
├─ 黃色: 中風險 (5人)
│  └─ 「李先生: 肺炎風險↑, 建議檢查呼吸音」
└─ 紅色: 高風險 (1人)
   └─ 「王女士: 跌倒風險↑↑, 24小時加強監測」

場景2: 主動介入
行動:
├─ 黃色: 安排醫生檢查、增加營養補充、運動課程調整
├─ 紅色: 立即送醫 (預防性, 而非等待事件發生)
└─ 預期效果: 肺炎住院↓35%, 跌倒骨折↓40%

場景3: 家屬告知
通知:
├─ 「親愛的李家人: 李先生最近活動量↓, 我們加強了監測」
├─ 「您也可以多打電話慰問」
└─ 家屬參與度↑ (從「無感」到「有感」)
```

#### 創新評分
```
技術突破: ⭐⭐⭐⭐ (8/10)
├─ 機器學習應用, 但非全新演算法
├─ 關鍵在於「整合50個感測器資料」
└─ 全球已有3-4個類似研究(Mayo, Cleveland Clinic)

長期影響: ⭐⭐⭐⭐ (8/10)

├─ 衛福部可納入長照2.0品質指標

評審吸引力: ⭐⭐⭐⭐⭐ (9/10)
```
