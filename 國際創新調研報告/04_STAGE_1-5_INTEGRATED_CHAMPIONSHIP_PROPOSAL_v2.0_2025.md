## 執行摘要

### 核心願景
將新竹市政府主導的赤土崎多功能館(標案114A109，投資2,287萬)，打造為**東亞首個「科技家庭全齡支持生態系」**——整合長照、托育、青少年發展的智慧型樞紐，同時實驗**ESG企業整合、時間銀行、參與式預算**等三大政策創新。

### 三大致勝優勢

#### 1. 高度的政策對接性 ✓✓✓
- **中央長照2.0**: 赤土崎的日照中心直接銜接
- **財劃法修正**: 新竹市2025年額外293億預算，赤土崎是最高效配置方案
- **新竹市政策優先**: 赤土崎已進入標案階段(10/30開標)，可立即執行

#### 2. 普世共鳴 + 在地急迫性 ✓✓✓
- **竹科家庭的根本困境**: 年薪300萬卻「人生貧戶」(時間貧困、照護困境)
- **新竹市的獨特優勢**: 竹科聚落提供了全台最密集的**「企業ESG承諾者」與「志工資源」**
- **跨世代同步解決**: 一館解決了「失智照護、幼托困境、青少年支持」三大社會問題

#### 3. 技術創新與制度創新的完美融合 ✓✓✓
- **五階段系統設計**:
  - Stage 1-2: 國際最佳實踐案例 + 建築/技術創新
  - Stage 3: 三大政策制度設計(ESG、時間銀行、參與式預算)
  - Stage 4-5: 3D空間規劃與完整呈現

### 預期成果 (五年期)
- **服務覆蓋**: 1,500+竹科家庭
- **成本節省**: 政府預算節省NT$100M+ (相比分散建置)
- **社會價值**: 24,000小時年均社區互助時間 + 1,500+居民的民主參與
- **企業參與**: TSMC、MediaTek等竹科龍頭的10年ESG承諾

### Stage 3B: 時間銀行系統 (8,400字)

**核心創新**: 「三代互助時間銀行」

```
長者時間 (退休人生智慧)
├─ 手工藝教導、人生導師、陪伴服務
├─ 年均累積100-150 TC (時間學分)
└─ 兌換: 認知訓練、日照補助、社區活動

家長時間 (專業技能、運輸協力)
├─ STEM教育、財務素養、共育協力
├─ 年均累積80-120 TC
└─ 兌換: 托育補助、長親日照、家務協助

青少年時間 (課業互助、陪伴、數位技能)
├─ 課業輔導、幼兒陪伴、銀髮數位教導
├─ 年均累積60-100 TC
└─ 兌換: 課業輔導、升學協助、社區服務學分
```

**社會價值**:
- **年均社區互助時間**: 12,000筆交易 = 24,000小時
- **市場價值**: 相當於NT$2.4M的有償照護服務
- **節省政府預算**: 長照20-30% + 幼托10-15%

**會員規模**:
- Year 1: 50人(試點)
- Year 2-3: 300人(擴展)
- Year 4+: 800人(成熟)

### Stage 3C: 參與式預算制度 (8,300字)

**核心創新**: 「多元民主治理」

```
赤土崎年度NT$40M營運預算
├─ 固定成本(75% = NT$30M)
│  └─ 人事費、設施維護、安全合規(館方控制)
│
└─ 參與式預算(25% = NT$10M) ✓ 社區決定!
   ├─ 課程與服務 (45% = NT$4.5M)
   ├─ 設施改善 (35% = NT$3.5M)
   └─ 社區活動 (20% = NT$2M)
```

**民主機制**:
1. **提案徵集** (2月): 開放全社區提案
2. **審議討論** (3月): 專家評析、市民研討會
3. **投票表決** (4月): 線上+實體投票(目標投票率40-50%)
4. **執行監督** (5月-11月): 月度進度、中期檢視、年度評估
5. **反饋改進** (12月): 成果評估、隔年改進

**防止「精英捕獲」的七項措施**:
1. 多元代表性監督委員會 (9人代議制)
2. 資訊完全公開透明
3. 投票前的獨立資訊研討會
4. 加權投票制(時間銀行會員1.5倍權重)
5. 多輪投票制(初篩→細化討論→最終排序)
6. 強制「反對意見」陳述時間
7. 年度外部審計與評估

**民主參與成果** (Year 3目標):
- 投票率: 50%
- 參與人數: 6,000人
- 社區凝聚力: 參與者的後續社區參與率↑40%
