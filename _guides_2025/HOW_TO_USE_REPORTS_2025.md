## 🏗️ 如何利用報告進行建築優化

**如果想改變建築設計**:

1. **查閱Stage 4 (建築設計)**
   - 了解目前的樓層配置與功能分配
   - 檢查隔音等級、安全通道、無障礙設計

2. **查閱docs/design/ (服務流程)**
   - 確認新設計不會破壞現有動線

3. **重新計算成本** (查閱Document 11)
   - 如果改變面積，需要重算CAPEX
   - 如果改變樓層功能，需要重算OPEX

4. **驗證政策符合性**
   - 查閱docs/policy-analysis/ 確保符合建築法規
   - 確保符合長照2.0、公托標準

**建議流程**:
```
想法 → Stage 4查詢 → 初步設計 → Document 11重算成本
→ 政策檢查 → 建築師審核 → 修訂完成
```

## 💰 如何利用報告進行財務優化

**如果想調整財務模型**:

1. **基準文件**: Document 11 (必讀)
   - 這是所有財務決策的基礎
   - 包含詳細的收支拆解

2. **改變的步驟**:
   - **改變服務收費?** → 查「營收預測」部分，重算Year 1-5
   - **改變人力配置?** → 查「人力成本」部分，逐部門調整
   - **改變政府補助?** → 查「政府補助」部分，重算自給率
   - **改變融資額度?** → 查「銀行可貸性」部分，重新評估

3. **驗證新模型**:
   - 計算Year 1-5自給率 (目標>75% Year 1)
   - 檢查現金流 (避免月度虧損>NT$1M)
   - 驗證DSCR (第3年>20%)

**財務改變的影響分析**:
```
降低服務費
    ↓
收入減少 → 自給率下降 → 需要更多政府補助或企業贊助
    ↓
需要重新評估可行性

增加人力
    ↓
成本增加 → 需要更高收費或政府補助
    ↓
可能超過竹科家庭的承受能力
```

**如果想加入新的服務**:

1. **新服務的可行性研究**:
   - 查閱docs/service-analysis/ 確認市場需求
   - 查閱Document 05-07 找相關論文支撐

2. **空間安排**:
   - 查閱Stage 4 (建築設計) 確認是否有多餘空間
   - 如果沒有，需要重新規劃樓層配置

3. **財務影響**:
   - 估計新服務的營收
   - 估計新服務的人力成本
   - 重算Document 11的財務模型

4. **人力調配**:
   - 新服務需要多少人?
   - 現有29人編制是否需要調整?

**擴展服務範例**:
```
假設想加入「親職教育課程」:

1. 驗證需求 (查docs/service-analysis/)
2. 估計收費: NT$2,000/人/月 × 50人 = NT$100K/月
3. 估計人力: 1名講師 + 0.5名助手 = 1.5人 (NT$80K/月)
4. 估計淨利: NT$100K - NT$80K = NT$20K/月
5. 重算財務: 全年增收NT$240K, 年淨利增加NT$180K
6. 驗證空間: 4F是否有教室空間?
7. 簽核: 符合政策嗎? (查docs/policy-analysis/)
```

## 🌐 如何利用報告進行全台複製

**如果想複製到其他城市** (台北、台中、高雄):

1. **複製的四個步驟**:

   **Step 1: 城市特性分析**
   - 用docs/service-analysis/ 的方法論分析目標城市
   - 確認該城市是否有類似的「竹科家庭」
   - 估計目標市場人口規模

   **Step 2: 政策適配**
   - 查詢該城市的長照2.0執行狀況
   - 查詢該城市的托育倍增進度
   - 尋找該城市的「關鍵決策者」或「政治機遇」

   **Step 3: 財務調整**
   - 用Document 11的模型進行調整
   - 考慮該城市的人事成本差異 (台中/高雄通常較低)
   - 考慮該城市的服務費用差異

   **Step 4: 簽核與推進**
   - 準備城市版Executive Summary
   - 準備該城市版本的政策對接分析
   - 與當地政府、企業溝通

2. **複製提案的適應**:

   **保留不變** ✓
   - 建築設計邏輯 (垂直整合)
   - 政策創新設計 (ESG + 時間銀行 + 民主)
   - 技術系統 (IoT + AI + 綠建築)

   **需要調整** ⚠️
   - 建築規模 (依目標市場大小調整)
   - 人員配置 (依當地薪資水平調整)
   - 服務收費 (依當地消費水準調整)
   - 政府補助額度 (依當地財政狀況調整)

3. **複製版本的報告包**:
