### Phase 1 的關鍵成功因素

```
成功的定義:
✅ 黑客松冠軍 (名聲 + 政治支持)
✅ 政府納入整合方案 (制度保障)
✅ 企業簽約 (資金保障)
✅ 設計單位就位 (執行保障)
✅ 初步設計完成 (技術保障)

失敗的風險:
❌ 簡報失利 → 影響: 5個月延遲, 政府信心下降
❌ 企業撤資 → 影響: 預算缺口, 融資困難
❌ 設計單位不願合作 → 影響: 改請其他單位, 需要2-3個月
❌ 政府換人 → 影響: 政策轉向, 可能中止

風險應對:
1. 多聯繫2-3家設計公司 (備選)
2. 與企業簽署「退出條款」保護政府
3. 與市長/議會建立直接關係
4. 備有「簡化版」方案 (假設資源不足)
```

#### 複製成功的關鍵

```

   └─ 包括設計、財務、運營、人才培養

2. 複製前必須有「赤土崎導師團隊」
   └─ 館長、技術人員、運營人員派駐支援新館

3. 複製前必須確保「資金與政策支持」
   └─ 不能因為赤土崎成功就自動獲得其他城市支持

4. 複製過程必須保留「文化基因」
   └─ 不是簡單搬用建築，而是複製「社福創新精神」

5. 複製後必須持續「經驗反饋」
   └─ 新館學到的東西反饋改進赤土崎與其他館
```

## 💡 進階開發的關鍵決策點

### 決策點1: 技術投資的深度 (2026年)

```
選項A: 輕量化技術版本
├─ 只導入IoT基礎監測
├─ 取消AI投影系統
├─ 簡化BMS能源管理
├─ 成本: -NT$10M, 但功能大打折扣

選項B: 完整技術方案 (建議)
├─ 全部導入 (IoT+AI+綠建築+BMS)
├─ 成本: NT$36.7M (計畫內)
├─ 優勢: 技術領先性, 吸引媒體與研究者

決策建議:
→ 選B, 因為:
  1. 技術投資是「複製差異化」的關鍵
  2. TSMC、MediaTek願意投資, 所以成本不是障礙
  3. 技術亮點能製造媒體聲量, 幫助複製
  4. 15年內, 技術成本會持續下降, 複製時會更便宜
```

### 決策點2: 政府補助政策 (2027年)

```
選項A: 向政府爭取「永久補助」
├─ 目標: Year 1-5 每年NT$6M
├─ 風險: 政治不穩定, 補助可能被刪
├─ 優勢: 現金流更穩定

選項B: 快速達成自給率, 減少補助依賴 (建議)
├─ 目標: Year 3達到70%自給, Year 5達到100%
├─ 優勢: 獨立性高, 複製時容易說服其他市政府
├─ 風險: Year 1-2現金流可能有壓力

決策建議:
→ 選B, 因為:
  1. 目標已經可達 (財務模型驗證過)
  2. 企業贊助可作為buffer
  3. 獨立性對複製至關重要
  4. 政府會更願意支持「自給館」而非「補助館」
```

### 決策點3: 國際合作方向 (2028年)

```
選項A: 國內複製為先, 國際輸出為後
├─ 時程: 國內先複製5-10個館 (2028-2035)
├─ 然後才考慮國際輸出 (2035+)
├─ 優勢: 發展完整, 經驗充分

選項B: 同步進行國內複製與國際輸出
├─ 時程: 2028年開始接待國際考察
├─ 2030年開始輸出到東南亞/東亞
├─ 優勢: 更高的國際影響力, 更多融資機會

決策建議:
→ 選B (混合模式), 因為:
  1. 國際聲譽有助於國內複製的政治支持
  2. 國際資金可補充國內融資不足
  3. UN/OECD認可有助於提升台灣軟實力
  4. 時機: 2028年開始接待考察是適當的
```
