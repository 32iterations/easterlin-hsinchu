## 🎯 方案A：轉換為 PowerPoint (.pptx)

### 方法1：使用 html2pptx 轉換器（推薦）

**前置需求**：
- Node.js 16+
- npm

**安裝步驟**：

```bash
# 1. 安裝 html2pptx 和 PptxGenJS
npm install html2pptx pptxgenjs

# 2. 運行轉換腳本
node convert-to-pptx.js
```

轉換腳本會自動：
- 讀取10個HTML檔案
- 轉換為PowerPoint格式
- 輸出 `hackathon-presentation.pptx`

### 方法2：手動製作 PowerPoint（備用）

如果無法使用轉換工具，可以：

1. 用瀏覽器打開每個HTML檔案
2. 截圖（Windows: Win+Shift+S / Mac: Cmd+Shift+4）
3. 貼到PowerPoint中，每頁對應一個幻燈片
4. 手動輸入文字內容（參考HTML中的文字）

**優點**：
- 不需要安裝任何工具
- 完全掌控版面
- 可以微調細節

**缺點**：
- 耗時約1-2小時
- 需要手動輸入文字

## 🌐 方案B：使用 reveal.js 互動式簡報

reveal.js 版本的簡報已創建在 `index.html`。

**使用方式**：

```bash
# 1. 啟動本地服務器
cd presentation
python -m http.server 8000

# 2. 打開瀏覽器訪問
http://localhost:8000/index.html

# 3. 簡報操作
# - 方向鍵：前後翻頁
# - F鍵：全螢幕
# - S鍵：演講者模式（顯示筆記和下一頁預覽）
# - Esc鍵：鳥瞰模式（查看所有幻燈片）
```

**優點**：
- 專業的過場動畫
- 支援演講者模式（筆記、計時器）
- 可以嵌入互動式圖表
- 可以匯出PDF（瀏覽器列印功能）

**缺點**：
- 需要瀏覽器播放（不是.pptx檔案）
- 比賽現場可能沒有網路（需離線版本）
