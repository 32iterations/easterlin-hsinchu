# 竹科家庭生活痛點資料分析專案

## 專案概述

本專案收集並分析了關於新竹科學園區家庭生活痛點的各類資料來源，包括新聞報導、社群討論、個人投書等。透過自動化處理與分類，提取出結構化資訊，幫助理解竹科家庭面臨的主要挑戰。

## 資料來源統計

### 總文章數：19 篇

**來源類型分佈：**
- 社群討論 (Mobile01)：10 篇 (52.6%)
- 新聞媒體：5 篇 (26.3%)
  - 風傳媒、鏡週刊、ETtoday、聯合新聞網
- 個人投書：3 篇 (15.8%)
- 其他：1 篇 (5.3%)

## 主題分析

### 前五大痛點

1. **工作-家庭平衡** (89.5%, 17篇)
   - 長工時、加班文化
   - 輪班制度影響家庭作息
   - 無法陪伴孩子與家人
   - 責任制工作型態

2. **財務壓力** (78.9%, 15篇)
   - 高房價與生活成本
   - 年收300萬仍感貧窮
   - 消費水平高

3. **生活品質** (68.4%, 13篇)
   - 生活機能不足
   - 娛樂選擇少
   - 風大、天氣因素
   - 交通壅塞

4. **跨縣市通勤** (57.9%, 11篇)
   - 與台北的往返困難
   - 大眾運輸不便
   - 需要家庭雙車

5. **托育/教育壓力** (42.1%, 8篇)
   - 托育機構時間限制
   - 教育資源競爭
   - 才藝學習壓力

### 其他議題

- **社群焦慮/教養競爭** (31.6%, 6篇)
  - 家長群組文化
  - 教養比較心態
  - 高端人口標籤

- **長照/失智照護** (15.8%, 3篇)
  - 雙薪家庭照顧長輩困難

- 2022年3月為討論高峰（Mobile01 討論串）
- 2025年討論再度升溫，顯示問題持續存在

## 關鍵發現

### 1. 高薪不等於幸福
年收入 300 萬的竹科家庭仍面臨：
- 時間貧窮：無暇陪伴家人
- 健康貧窮：身心健康亮紅燈
- 生活貧窮：缺乏正常家庭時光

### 2. 工作文化問題
- 早上 6-7 點到公司，晚上 9-10 點下班
- 輪班制度（凌晨 5 點值早班）
- 假日加班頻繁（每月至少一週）

### 3. 托育困境
- 托育機構營業時間無法配合加班
- 雙薪家庭接送孩子困難
- 考績受影響（需提早接孩子）

### 4. 地域適應問題
- 台北人難適應新竹生活
- 生活機能與娛樂選擇落差
- 大眾運輸系統不足

### 5. 社會壓力
- 家長群組高門檻
- 教養競爭激烈
- 物質比較風氣

## 產出檔案說明

### 1. `structured_data.json`
結構化資料，包含每篇文章的：
- 標題 (title)
- 發布日期 (date)
- 來源類型 (source_type)
- 來源檔案 (source_file)
- 內文摘要 (content, 限2000字)
- 關鍵詞 (keywords, 自動提取)
- 主題分類 (topics)

### 2. `topic_distribution.csv`
主題分佈統計表：
- 各主題文章數量
- 佔總文章比例

### 3. `timeline.csv`
時間序列分析：
- 按月統計文章數量
- 了解討論熱度變化

## 技術說明

### 使用工具
- **BeautifulSoup4**：HTML 解析
- **jieba**：中文分詞與關鍵詞提取
- **Python 3.13**

### 處理流程
1. 掃描目錄中所有 HTML 和 Markdown 檔案
2. 提取標題、日期、內文
3. 移除廣告、導航等非內容元素
4. 使用 jieba TF-IDF 提取關鍵詞
5. 基於關鍵詞匹配進行主題分類
6. 輸出結構化 JSON 與統計 CSV

### 執行方式

```bash
# 安裝依賴
pip install -r requirements.txt

# 執行處理腳本
python process_articles.py
```

## 結論

竹科家庭雖然擁有高收入，但面臨的生活挑戰是多維度的：

1. **時間維度**：長工時壓縮家庭時間
2. **空間維度**：生活機能與娛樂不足
3. **社會維度**：教養競爭與群組壓力
4. **健康維度**：身心健康受損
5. **財務維度**：高房價與消費水平

這些痛點不是單一因素造成，而是工作文化、城市發展、社會風氣等多重因素交織的結果。需要從政策、企業文化、社會氛圍等多方面著手改善。
