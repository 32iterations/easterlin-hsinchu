## 📊 第一部分：當前簡報的致命缺陷

### 🔴 嚴重問題 (Critical Issues)

#### 1. **比例過時 (4:3 vs 16:9)**
```
當前: 10" × 7.5" = 4:3 比例
問題:
  ❌ 現代投影機/螢幕 99% 都是 16:9
  ❌ 畫面兩側會出現黑邊，顯得不專業
  ❌ 浪費 30% 的螢幕空間
  ❌ 給人「2010年代」的過時感

標準: 13.333" × 7.5" = 16:9 比例
```

#### 2. **文字密度過高 (Text Overload)**
```
當前狀況:
  • Slide 2-3: 每張約 150-200 字
  • Slide 6: 建築規劃塞了 250+ 字
  • 字體大小: 13-16pt (太小！)

問題:
  ❌ 觀眾無法在 3 秒內掌握重點
  ❌ 違反「6x6 規則」(每張最多 6 行，每行最多 6 字)
  ❌ 報告人會「照本宣科」，失去說服力
  ❌ 投影到大螢幕後，後排看不清楚

標準:
  ✓ 每張投影片: 50-80 字以內
  ✓ 標題: 32-44pt
  ✓ 內文: 20-28pt
  ✓ 重點數字: 60-80pt
```

#### 3. **缺乏視覺層次 (No Visual Hierarchy)**
```
當前問題:
  ❌ 所有元素都是「文字方塊」
  ❌ 沒有圖表、圖示、流程圖
  ❌ 顏色單一（只有標題有顏色）
  ❌ 缺乏視覺引導（眼睛不知道先看哪裡）

觀眾反應:
  • 5 秒: 「這張好多字...」（開始分心）
  • 10 秒: 「算了，等報告人講吧」（放棄閱讀）
  • 30 秒: 開始滑手機
```

#### 4. **數據呈現方式錯誤 (Poor Data Visualization)**
```
當前: SROI 計算用「表格」呈現
問題:
  ❌ 表格需要「逐行閱讀」，認知負擔高
  ❌ 無法快速比較數值大小
  ❌ 缺乏視覺衝擊力

最佳實踐:
  ✓ 用「圓餅圖」顯示佔比（31% / 41% / 21% / 7%）
  ✓ 用「大數字」凸顯 SROI = 1:2.04
  ✓ 用「進度條」顯示目標達成率
  ✓ 用「信息圖」(Infographic) 說故事
```

## 🎯 第二部分：2025年企業ESG簡報最佳實踐

### ✅ 國際標準 (Based on McKinsey, BCG, Deloitte)

#### 1. **一頁一概念原則 (One Slide, One Idea)**
```
❌ 錯誤範例: Slide 2 塞了「三大挑戰」
   → 觀眾記不住，報告人講得很趕

✓ 正確做法:
   Slide 2: 挑戰概覽（3 個關鍵字 + icon）
   Slide 3: 挑戰 1 深入說明（1 張圖表 + 3 bullet points）
   Slide 4: 挑戰 2 深入說明
   Slide 5: 挑戰 3 深入說明

結果: 從 1 張變 4 張，但說服力 +300%
```

#### 2. **金字塔原則 (Pyramid Principle)**
```
結構:
┌──────────────────────┐
│   主論點 (大標題)      │  ← 1 秒看懂
├──────────────────────┤
│   支持論點 1          │
│   支持論點 2          │  ← 3 秒掃描
│   支持論點 3          │
├──────────────────────┤
│   證據 (圖表/數據)     │  ← 10 秒理解
└──────────────────────┘

當前簡報: 倒金字塔（先講細節，再講結論）❌
```

#### 3. **3秒法則 (3-Second Rule)**
```
測試: 給陌生人看投影片 3 秒，問「這張在講什麼？」

當前簡報測試結果:
  Slide 2: 「呃...挑戰？很多字...」 ❌
  Slide 6: 「建築？樓層？看不懂...」 ❌
  Slide 8: 「表格...數字...」 ❌

理想回答:
  Slide 2: 「台積電面臨三大人才挑戰」 ✓
  Slide 6: 「一棟樓整合四種服務」 ✓
  Slide 8: 「投資1元，回報2元」 ✓
```

#### 4. **80/20 視覺法則**
```
比例分配:
  20% 文字
  80% 視覺元素（圖表、圖示、照片、留白）

當前簡報:
  90% 文字 ❌
  10% 視覺元素（而且只是彩色文字框）❌
```

