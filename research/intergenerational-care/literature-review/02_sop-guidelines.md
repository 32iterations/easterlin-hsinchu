# 代际照护场馆 SOP 实务指引

**基于**: 2020-2025学术实证综合
**适用场域**: 共址托婴中心+长照日照中心+青少年活动空间
**更新日期**: 2025-10-23

---

## 🏥 一、通风与感染控制 SOP

### 1.1 三阶段通风策略

参考：**Hayashi et al. (2024) 日本医院COVID爆发调查**

| 阶段 | 触发条件 | 通风要求 | 监测指标 | 执行动作 |
|------|----------|----------|----------|----------|
| **A. 常态运作** | - 社区无疫情<br>- 馆内无症状个案 | • 换气率 **6-8次/小时**<br>• CO2 **<800ppm**<br>• 温度 22-26°C<br>• 湿度 40-60% | • CO2即时监测<br>• 每日巡检纪录 | • 定期维护空调设备<br>• 每月更换滤网<br>• 每日记录CO2峰值 |
| **B. 爆发风险** | - 社区传播率↑<br>- 邻近机构群聚<br>- 馆内出现疑似症状 | • 换气率 **10-12次/小时**<br>• CO2 **<700ppm**<br>• 增设**HEPA便携式滤网** | • CO2 **<700ppm**<br>• 每4小时巡检<br>• 症状日报表 | • 加强自然通风(定时开窗)<br>• 启动HEPA机组<br>• 缩减共享空间容留人数 |
| **C. 收治感染者**<br>(或隔离观察) | - 确诊个案<br>- 密切接触者隔离 | • 隔离区换气率 **≥12次/小时**<br>• **负压**(可行时)<br>• UVGI消毒启动<br>• 气流控制(清洁→污染) | • CO2持续监测<br>• 压差监测(-2.5Pa)<br>• HEPA滤网**每周**更换 | • 隔离区独立空调<br>• 暂停代际互动<br>• 改为纯线上服务<br>• 全员PCR/RAT筛检 |

### 1.2 成本效益工程控制优先级

参考：**Bueno de Mesquita et al. (2021)**

| 措施 | 气溶胶风险降低 | 成本 | 实施难度 | 优先级 | 备注 |
|------|---------------|------|----------|--------|------|
| **便携式HEPA滤网机** | 80%+ | **低-中**<br>(NT$2-5万/台) | ⭐ | ⭐⭐⭐ | 立即可用、弹性布署 |
| **CO2监测器全区部署** | 间接指标 | **低**<br>(NT$5千-1万/台) | ⭐ | ⭐⭐⭐ | 必备基础设施 |
| **定时自然通风SOP** | 40-60% | **零** | ⭐ | ⭐⭐⭐ | 每2小时开窗10分钟 |
| 提升换气率(改造风管) | 80%+ | **高**<br>(NT$100-300万) | ⭐⭐⭐ | ⭐⭐ | 建造期一次到位 |
| 中央空调UVGI升级 | 60-70% | **高**<br>(NT$80-150万) | ⭐⭐ | ⭐⭐ | 需专业维护 |
| 负压隔离室改造 | >90% | **极高**<br>(NT$200-500万/间) | ⭐⭐⭐ | ⭐ | **仅隔离观察区** |

**预算建议**（赤土崎馆1500坪场域）：
- 基础配置：HEPA x10台 + CO2监测x20台 = **NT$ 70万**
- 标准配置：基础 + 风管改造 = **NT$ 250万**
- 完整配置：标准 + UVGI + 负压隔离室 = **NT$ 550万**

### 1.3 空气质量监测基准

| 指标 | 常态标准 | 警戒值 | 立即行动值 | 监测频率 |
|------|---------|--------|-----------|----------|
| **CO2浓度** | <800ppm | 800-1000ppm | >1000ppm | 即时监测(每5分钟) |
| **温度** | 22-26°C | 26-28°C / 20-22°C | >28°C / <20°C | 每小时 |
| **相对湿度** | 40-60% | 60-70% / 30-40% | >70% / <30% | 每小时 |
| **PM2.5** | <35μg/m³ | 35-55μg/m³ | >55μg/m³ | 每小时 |

**超标应对流程**：
1. **警戒值** → 检查通风设备、增加开窗频率
2. **立即行动值** → 疏散至其他空间、启动备用HEPA、设备检修

## 🏗️ 二、分舱与空间规划 SOP

### 2.1 功能分区原则

```
┌────────────────────────────────────────────────────────┐
│                    清洁 → 缓冲 → 活动 → 隔离             │
├────────────────────────────────────────────────────────┤
│  【清洁区】      【缓冲区】       【活动区】    【隔离区】│
│  办公室/储藏  ←→ 更衣/洗手站 ←→ 托婴/日照  ←→ 观察室   │
│                                  /青少年      (负压)    │
│      ↓             ↓               ↓             ↓      │
│  严格单向气流：清洁 → 半清洁 → 潜在污染 → 确诊污染    │
└────────────────────────────────────────────────────────┘
```

**气流方向控制**：
- 清洁区：正压 +5Pa
- 活动区：中性 ±0Pa
- 隔离区：负压 -2.5Pa

- 高互动区：150坪（容纳30-40人同时段）
- 低互动区：80坪（观察廊道+展示区）
- 独立活动区：托婴200坪 + 日照150坪 + 青少年100坪

### 2.3 清洁消毒周期

参考：**日本厚生劳动省托育设施指引（2023修订）**

| 区域/物品 | 消毒剂 | 频率 | 负责人 | 记录表 |
|----------|--------|------|--------|--------|
| **托婴玩具** | 次氯酸钠200ppm | **每日1次**<br>(放学后) | 保育员 | ✓ |
| **共享中庭/活动室** | 75%酒精<br>或次氯酸水 | **每日2次**<br>(午餐后、闭馆前) | 清洁人员 | ✓ |
| **电梯按钮/扶手/门把** | 75%酒精 | **每4小时** | 驻点人员 | ✓ |
| **餐桌/椅子** | 次氯酸钠100ppm | **每次使用后** | 使用单位 | ✓ |
| **厕所/洗手台** | 漂白水500ppm | **每2小时** | 清洁人员 | ✓ |
| **空调滤网** | 清洗 | **每月1次**<br>(阶段B为每2周) | 设施组 | ✓ |
| **HEPA滤网** | 更换 | **每季1次**<br>(阶段C为每周) | 设施组 | ✓ |

**消毒标准作业**：
1. 先清洁(移除污垢) → 再消毒
2. 消毒后需**自然干燥**或清水擦拭（避免残留刺激呼吸道）
3. 所有消毒纪录保存≥1年

## 👥 三、互动时长与频率 SOP

### 3.1 实证最佳方案

基于：**Lee et al. (2025); Whear et al. (2023); Campbell et al. (2024)**

| 介入类型 | 时长 | 频率 | 持续期 | 证据等级 | 预期效果 |
|----------|------|------|--------|----------|----------|
| **🎵 结构化音乐活动** | **60-90分钟** | **每周2次** | **≥6个月** | ⭐⭐⭐ | 老人：愉悦感↑、躁动↓<br>幼儿：社交能力↑ |
| **🌱 参与式园艺** | **60-90分钟** | **每周2次** | **≥6个月** | ⭐⭐⭐ | 老人：认知功能↑、活动量↑<br>幼儿：环境意识↑ |
| **📖 怀旧/故事时光** | **45-60分钟** | **每周2次** | **≥6个月** | ⭐⭐ | 老人：归属感↑、语言刺激↑<br>幼儿：语言发展↑ |
| **🎨 艺术创作** | **60-90分钟** | **每周1-2次** | **≥6个月** | ⭐⭐ | 双向：创意表达↑、成就感↑ |
| **自由互动访视** | **30-45分钟** | **每周1次** | 不限 | ⭐ | 短期效果、需持续 |

**⚠ 关键发现**：
- **定期性 > 单次时长**（每周2次 × 60分钟 > 每周1次 × 120分钟）
- **1年期深度互动** 才显著改善福祉（Campbell et al., 2024）
- **老幼配对黄金组合**：5-8岁儿童 + 65-80岁认知健全老人

## 🩺 四、适应症与禁忌症 SOP

### 4.1 参与者筛选标准

#### **老年族群**

| 类别 | 标准 | 评估工具 | 评估频率 |
|------|------|----------|----------|
| **✓ 适应症** | • MMSE **>5分**(轻中度认知障碍)<br>• 无急性感染<br>• 可维持30分钟+坐姿或参与<br>• 无攻击性行为史(过去6个月) | • MMSE<br>• ADL量表<br>• 行为观察纪录 | • 初次评估<br>• 每季追踪 |
| **△ 相对禁忌**<br>(需个别评估) | • MMSE **5-10分**(中重度失智)<br>• 轮椅使用者<br>• 听力/视力严重障碍<br>• 呼吸道症状恢复期 | 同上 + 医师诊断 | 每次活动前 |
| **✗ 绝对禁忌** | • 传染病确诊期<br>• 发烧>38°C<br>• 严重免疫低下(化疗/器官移植)<br>• 暴力倾向(攻击他人纪录) | 医疗纪录<br>+ 家属告知 | **每日晨检** |

**特殊状况处理**：
- MMSE <5分者 → 仅参与**低互动区**观察式活动
- 轮椅使用者 → 参与**静态活动**(音乐聆听、艺术欣赏)，避免需移动活动

---

#### **儿童/青少年族群**

| 年龄层 | ✓ 适应症 | ✗ 绝对禁忌 | 家长同意 |
|--------|----------|-----------|----------|
| **0-2岁婴幼儿** | • 完成月龄应有疫苗<br>• 无急性感染 | • 发烧、呼吸道症状<br>• 肠胃炎(腹泻/呕吐) | **必须签署**<br>代际互动同意书 |
| **3-5岁幼儿** | • 可理解基本指令<br>• 完成常规疫苗<br>• 无严重过敏 | 同上 +<br>• 水痘发疹期<br>• 手足口病 | **必须签署** |
| **6-11岁儿童** | • 可遵守活动规则<br>• 愿意与长者互动 | 同上 | **必须签署** |
| **12-18岁青少年** | • 自愿参与(非强制)<br>• 经基础训练(如何与失智老人互动) | 传染病确诊 | **建议签署**<br>(12岁+可自主同意) |

### 4.2 每日健康筛检 SOP

**时间**: 每日活动前30分钟
**负责人**: 各单位护理人员/保育员

#### **筛检流程**

```
┌─────────────────────────────────────────┐
│ Step 1: 体温测量 (额温枪/耳温枪)          │
│   → <37.5°C 通过 | ≥37.5°C 复检 | ≥38°C 隔离 │
├─────────────────────────────────────────┤
│ Step 2: 症状询问 (使用标准化问卷)         │
│   □ 咳嗽 □ 流鼻水 □ 喉咙痛                 │
│   □ 呕吐/腹泻 □ 皮疹 □ 呼吸困难           │
│   → 任一项✓ → 隔离观察 + 医疗评估        │
├─────────────────────────────────────────┤
│ Step 3: 行为观察 (老人特别注意)           │
│   • 精神活力 • 食欲 • 睡眠品质             │
│   → 异常变化 → 记录追踪                  │
├─────────────────────────────────────────┤
│ Step 4: 记录与通报                        │
│   • 填写健康筛检表                        │
│   • 异常者→24小时追踪→家属通知           │
└─────────────────────────────────────────┘
```

**筛检纪录保存期限**: 3年

---

### 4.3 环境禁忌症（立即停止活动）

| 环境指标 | 停止标准 | 应对措施 | 恢复条件 |
|----------|----------|----------|----------|
| **CO2浓度** | >1000ppm | • 疏散至其他空间<br>• 全面开窗通风<br>• 检查通风设备 | CO2 <800ppm<br>+ 设备正常运转 |
| **温度** | >28°C 或 <20°C | • 调整空调<br>• 冬季加开暖气<br>• 夏季启动备用冷气 | 恢复22-26°C |
| **群聚感染** | 同时段≥3人出现症状 | • **立即停止所有代际活动**<br>• 环境全面消毒<br>• 接触者追踪14天 | 无新增病例7天<br>+ 环境检测合格 |
| **设备故障** | 通风/空调完全失效 | • 转移至户外<br>• 或延期活动 | 设备修复+测试 |

**观察重点**：

## 🚨 六、紧急应变 SOP

### 6.1 疑似传染病爆发应变流程

```
┌─────────────────────────────────────────────────┐
│ 触发条件：24小时内≥2人出现相似症状              │
├─────────────────────────────────────────────────┤
│ Step 1: 立即隔离 (5分钟内)                       │
│   • 引导症状者至隔离观察室                       │
│   • 配戴医用口罩                                │
│   • 记录接触者清单                               │
├─────────────────────────────────────────────────┤
│ Step 2: 通报启动 (15分钟内)                      │
│   • 通知馆长/主任                               │
│   • 联络卫生局疾管科                            │
│   • 通知家属                                    │
├─────────────────────────────────────────────────┤
│ Step 3: 环境管制 (30分钟内)                      │
│   • **停止所有代际活动**                        │
│   • 发病区域封锁                                │
│   • 启动阶段C通风模式                           │
│   • 全面环境消毒（漂白水500ppm）                │
├─────────────────────────────────────────────────┤
│ Step 4: 接触者追踪 (3小时内)                     │
│   • 调阅监视器/签到表                           │
│   • 匡列密切接触者（接触>15分钟且<1公尺）       │
│   • 通知接触者进行健康监测                      │
│   • 建议RAT/PCR筛检                             │
├─────────────────────────────────────────────────┤
│ Step 5: 持续监测 (14天)                          │
│   • 每日2次体温回报                             │
│   • 症状日志填写                                │
│   • 新增病例→重启流程                          │
└─────────────────────────────────────────────────┘
```

**解除条件**：
- 无新增病例连续7天 + 环境检测合格 → 恢复阶段A运作
- 恢复首周采阶段B通风标准（保守）
