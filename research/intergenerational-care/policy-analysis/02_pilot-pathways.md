# 不修法的試辦路徑：新竹市赤土崎館實戰操作手冊

**文件用途**：提供可立即啟動的試辦方案，無需等待中央修法
**核心策略**：「行政契約」+「採購創新」+「企業夥伴」三管齊下
**時間窗口**：標案10/30開標後，規劃設計階段是納入功能的最後機會

---

## 🎯 試辦路徑總覽

### 核心邏輯：用「契約自由」繞過「法規僵化」

台灣現行法規的三大特性：
1. **禁止性規範**（不能做的）→ 必須遵守
2. **授權性規範**（可以做的）→ 有彈性空間
3. **法律空白**（沒規定的）→ **行政創新空間**

**試辦策略**：
- 禁止性規範：嚴格遵守（如師生比、空間標準）
- 授權性規範：往友善方向解釋（如「臨時托育」涵蓋夜間）
- 法律空白：大膽創新（如企業MOU、跨局處預算池、家長諮議機制）

## 📍 路徑一：赤土崎館整合服務試辦

### 1.1 法律依據（不需修法）

| 項目 | 法源 | 創新詮釋 |
|------|------|---------|
| **共址整合** | 兒童及少年福利機構設置標準 第3條 | 「同一建築物內可設置多種福利機構」→ 托嬰中心+日照中心共址合法 |
| **臨時托育** | 托嬰中心管理辦法 第7條 | 「臨時托育」未限定時段 → 可涵蓋夜間/假日 |
| **服務創新** | 地方制度法 第18-19條 | 地方政府自治事項 → 可自訂服務模式 |
| **採購彈性** | 政府採購法 第22條第1項第9款 | 「具有特殊性質」→ 可採限制性招標或最有利標 |

#### 一、服務規格（必要條件）
1. 托嬰中心：收托0-2歲40人，師生比1:5，通過評鑑甲等以上

3. 【企業EAP整合】（佔評分15%）
   - 與竹科企業HR系統串接預約功能

5. 【家長參與治理】（佔評分10%）
   - 設立「竹科家長顧問團」（每季至少開會1次）

   - 建立滿意度即時回饋機制（每月調查）

#### 三、財務條件
- 市府補助：營運成本60%（托嬰+日照依現行補助標準）
- 使用者付費：30%（分級收費，低收減免）
- 企業認養/捐贈：10%（投標廠商需提出企業募資計畫）

#### 四、履約期間
- 第一期：3年（115-117年）

**關鍵創新**：
- ✅ 用「評選加分」取代「強制要求」（避免流標）

- ✅ 財務三軌（政府+用戶+企業），降低市府負擔

## 📍 路徑二：竹科企業彈性工時自願聯盟

### 2.1 法律依據（不需修法）

台灣現行法規**不禁止**企業自主提供彈性工時：
- 勞動基準法第30條：「勞工正常工作時間，每日不得超過八小時，每週不得超過四十小時」→ 規範上限，不禁止彈性安排
- 性別工作平等法第19條：「受僱者為撫育未滿三歲子女，得向雇主請求調整工作時間」→ 僅適用3歲以下，但不禁止企業自願擴大
- **契約自由原則**：企業與員工可在不違反強制規定前提下，自由約定優於法律的條件

### 2.2 「竹科家庭友善企業聯盟」操作方案

#### 成立時程
- **115年1月**：新竹市政府勞工處召集說明會（邀請10-15家竹科企業）
- **115年3月**：簽署「竹科家庭友善宣言」（第一批5家企業）
- **115年6月**：建立「友善企業認證標章」制度
- **116年起**：每年擴大聯盟成員

#### 彈性工時自願承諾清單（企業自選，非強制）

| 彈性措施 | 內容 | 適用對象 | 參考企業實例 |
|---------|------|----------|------------|
| **A1. 彈性上下班** | 09:00-10:00彈性到班、17:00-18:00彈性離班 | 所有員工 | Google、微軟 |
| **A2. 壓縮工時** | 每週工作4天（每日10小時），3天休息 | 研發/行政人員 | Salesforce |
| **A3. 在家工作** | 每週至少1天可在家工作 | 非產線人員 | 台積電（疫情期間已試行） |
| **A4. 學期工作制** | 配合孩子學期上課（寒暑假可減少工時） | 有學齡子女員工 | 英國公部門 |
| **A5. 育兒時段** | 每日可請1小時「育兒假」（接送小孩，不扣薪） | 有0-6歲子女員工 | 聯合利華 |
| **A6. 緊急家庭假** | 每年5天有薪「緊急家庭照顧假」（小孩/長輩突發狀況） | 所有員工 | 花旗銀行 |

**聯盟承諾分級**（企業可選擇加入等級）：

```
🥇 金牌友善（6項措施全採用）
   └→ 市府採購加3%評分、媒體宣傳、市長頒獎

🥈 銀牌友善（4-5項措施）
   └→ 市府採購加2%評分、網站公開表揚

🥉 銅牌友善（2-3項措施）
   └→ 市府採購加1%評分、認證標章授權使用
```

#### 市府配套措施（無需修法）

1. **採購優惠**（政府採購法第52條允許「評分優惠」）
   - 友善企業投標市府採購案，可獲1-3%加分
   - 適用範圍：勞務採購、財物採購（如電腦設備、顧問服務）

2. **品牌宣傳**
   - 市府官網「友善企業專區」
   - 每年舉辦「竹科友善家庭論壇」
   - 媒體聯合報導（提升企業ESG評分）

3. **人才招募優勢**
   - 與清大/交大/陽明交大合作，優先推薦學生至友善企業
   - 市府就業中心優先媒合友善企業職缺

**關鍵創新**：
- ✅ 不增加市府總預算，只是「重新排列組合」
- ✅ 打破局處本位主義（「這是社會處的案子，關教育處什麼事」）
- ✅ 避免資源重複浪費（原本三個館要編3倍預算，現在整合只要1.2倍）

#### 行政程序（不需議會特別授權）

1. **市長指示**：召集社會處、教育處、衛生局協商
2. **簽訂行政契約**：三局處簽署「赤土崎館共同營運協議書」
3. **預算科目調整**：各局處115年度預算編列時，增列「赤土崎館分攤款」
4. **議會備查**：向市議會報告整合效益（通常不會反對，因為總額沒增加）

### 3.2 家長參與治理：「竹科家長顧問團」

#### 法律依據（不需修法）

- 地方制度法第16條：「地方政府得設各種委員會」→ 可自主設立諮議組織
- 行政程序法第165條：「行政機關得以協議方式，與民眾建立協力關係」

#### 組織設計

```
竹科家長顧問團（15-20人）
├─ 組成：
│  ├─ 赤土崎館使用者家長：8人（抽籤選出）
│  ├─ 竹科企業HR代表：3人（輪值）
│  ├─ 社福/教育/醫療專家：3人（市府邀請）
│  └─ 市府代表（列席）：社會處長、教育處長、衛生局長
│
├─ 任期：2年（可連任1次）
│
├─ 運作：
│  ├─ 每季召開1次定期會議
│  ├─ 重大決策（收費調整、課程變更）需顧問團同意
│  └─ 提供市府政策建議（如擴大夜間托育、增設臨托點）
│
└─ 權責：
   ├─ 審議權：課程內容、收費標準、營運時間
   ├─ 監督權：查核營運單位績效、調閱滿意度數據

```

## ⚠️ 風險管理與應變方案

### 風險一：建築師不願調整設計

**應變方案**：
1. 強調「整合設計」是「加分項」，有助於得獎/媒體曝光
2. 提供國際案例參考（新加坡Kampung Admiralty獲獎無數）
3. 若堅持不改，則啟動「變更設計」程序（可能增加200-300萬費用，但值得）

### 風險二：企業不願簽署MOU

**應變方案**：
1. 先從「最友善企業」下手（聯發科、台達電ESG評分高）
2. 降低門檻：年捐款50萬也可加入聯盟
3. 強調媒體效益：「全國第一家簽署友善宣言的半導體企業」

### 風險三：營運委託案流標

**應變方案**：
1. 降低「夜間臨托」要求：從「必要條件」改為「加分項目」
2. 提高政府補助比例：從60%提高到70%
3. 開放「聯合投標」：允許托嬰中心+日照中心+青少年組織聯合承攬

### 風險四：家長實際需求不如預期

**應變方案**：
1. 啟用前進行「需求調查」（竹科企業配合發問卷）
2. 採「動態調整」：如夜間臨托使用率低，可調整為「假日全日托」
3. 保留彈性：空間設計允許功能轉換（如臨托區可改為早療教室）

## 🎯 總結：為何不需要等修法？

### 台灣行政創新的三大空間

```
┌────────────────────────────────────────┐
│  空間一：地方自治事項（地方制度法）      │
│  ─────────────────────────────────    │
│  ✓ 社福服務模式：地方政府自主決定        │
│  ✓ 組織編制：可設專案小組/委員會         │
│  ✓ 預算運用：在總額內可跨局處調整         │
├────────────────────────────────────────┤
│  空間二：契約自由（民法/政府採購法）      │
│  ─────────────────────────────────    │
│  ✓ 企業MOU：雙方自願簽署，無需法律授權   │
│  ✓ 營運委託：採購規格可創新設計           │
│  ✓ 捐贈協議：企業CSR捐款屬自由意志        │
├────────────────────────────────────────┤
│  空間三：法律解釋彈性（行政程序法）       │
│  ─────────────────────────────────    │
│  ✓ 「臨時托育」可涵蓋夜間時段              │
│  ✓ 「共址整合」符合設置標準               │
│  ✓ 「家長參與」屬民眾協力行政             │
└────────────────────────────────────────┘
```
