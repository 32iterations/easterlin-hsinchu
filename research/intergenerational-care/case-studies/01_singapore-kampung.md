# 案例A：新加坡 Kampung Admiralty（共址整合标杆）

**设计团队**: WOHA Architects + Henning Larsen
**地点**: 新加坡北部Woodlands区
**基地**: 0.9公顷（约2,722坪）
**完工**: 2017年
**荣誉**: 2018年世界建筑节年度最佳建筑
**调研日期**: 2025-10-23

---

## 📐 一、空间配置：垂直村落（Vertical Kampung）三明治模型

### 1.1 整体结构

```
┌────────────────────────────────────────────────────┐
│  楼层 11-21F  │ 🏘️ 长者住宅（104户）            │ ← 上层
│               │ • 1房/2房公寓                    │
│               │ • 屋顶社区公园（1公顷）          │
│               │ • 垂直绿化、步道、休憩亭         │
├───────────────┼──────────────────────────────────┤
│  楼层 5-10F   │ 🏥 医疗中心                      │ ← 中层
│               │ • 全科诊所、专科门诊             │
│               │ • 日间照护中心                   │
│               │ • 中央天井采光                   │
│               │ • 候诊区面向绿化庭园             │
├───────────────┼──────────────────────────────────┤
│  楼层 1-4F    │ 🏢 社区广场（Community Plaza）   │ ← 底层
│  L4           │ • 多功能活动厅                   │
│  L3           │ • **活跃乐龄中心**(Active Aging Hub) │
│  L2           │ • **托婴中心**(Childcare)        │
│               │ • 小贩中心(Hawker Centre)        │
│               │ • 超市、商店、咖啡厅             │
│  L1(地面层)   │ • 完全开放、步行穿越             │
│               │ • 景观广场、座椅区               │
└────────────────────────────────────────────────────┘
```

**垂直整合逻辑**：
- **上层（住）**：长者居住，享有绿化公园
- **中层（医）**：医疗支持，方便住民就诊
- **底层（活）**：社区交流，老幼代际互动核心

---

### 1.2 关键空间设计特色

#### **中央天井（Central Void）**

```
              ┌─────────────┐
              │  天空层 11F  │ 屋顶公园
              └──────┬──────┘
                     │
              ┌──────┴──────┐
              │  天井贯穿     │ 自然采光
              │  5F → 11F    │ 自然通风
              └──────┬──────┘
                     │
              ┌──────┴──────┐
              │ 绿化中庭 5F   │ 候诊休憩
              └──────────────┘
```

**功能**：
- 自然采光减少人工照明（节能30%）
- 烟囱效应自然通风（CO2自然排出）
- 视觉穿透感，消除"机构感"

#### **托婴中心 + 乐龄中心紧邻配置**

**平面示意**（L2-L3楼层）

```
┌────────────────────────────────────────┐
│ L3: 活跃乐龄中心                        │
│  ┌─────────┐  ┌──────────┐            │
│  │ 活动室A  │  │ 共享中庭 │ ← 代际互动  │
│  │(音乐/艺术)│  │(玻璃帷幕)│    热点     │
│  └────┬────┘  └────┬─────┘            │
│       │ 楼梯       │                    │
├───────┴────────────┴────────────────────┤
│ L2: 托婴中心                            │
│  ┌─────────┐  ┌──────────┐            │
│  │ 幼儿教室 │  │ 共享中庭 │            │
│  │(3-5岁)   │  │(下方)    │            │
│  └─────────┘  └──────────┘            │
└────────────────────────────────────────┘
```

**设计要点**：
- ✓ **上下层关系**：透过玻璃地板/天窗可相互看见（视觉互动）
- ✓ **共享中庭**：定期活动时间，幼儿"上楼"至L3参与
- ✓ **独立出入口**：托婴有专用电梯，避免交叉感染

### 1.3 动线设计

#### **垂直连通**

| 使用者 | 主要动线 | 设计考量 |
|--------|----------|----------|
| **长者居民**(11-21F) | 住宅专用电梯x2 → L1景观广场<br>→ L2-3社区中心 | 电梯门厅宽敞(轮椅友善)<br>按钮高度降低至110cm |
| **医疗患者**(5-10F) | 专用电梯x2 → 医疗层<br>避开居民动线 | 可直达医疗层，不经过社区广场 |
| **托婴幼儿**(L2) | 地面层专用入口 → L2<br>托婴专用电梯x1 | 家长接送不必穿越人群 |
| **社区民众**(L1-4) | 步行穿越地面层 → 楼梯/电梯 | 完全开放，无门禁 |

#### **水平渗透**（地面层permeability设计）

```
公园绿地 ←→ L1开放广场 ←→ 小贩中心 ←→ 对街街区
         (步行可穿越)
```

**空间句法分析**（引自研究论文）：
- **半公共空间** 占47.62% → "Easy" permeability
- **导向清晰度** 高 → 老人不易迷路
- **社区融合** → 非住民也可使用小贩中心、公园

### 2.2 创新人力模式：内部志工经济

|---------|------|------|----------|
| **代际活动志工** | • 每周下楼至L2-3参与幼儿互动<br>• 故事时光、传统手工艺教学 | 每周1-2次<br>每次60分钟 | • 社区点数(兑换餐券)<br>• 年度表扬 |

## 🦠 四、感染控制

### 4.1 建筑设计层级

#### **被动式环境控制**（无需耗能）

| 策略 | 设计 | 效果 | 监测 |
|------|------|------|------|
| **中央天井自然通风** | 5F-11F贯穿天井<br>+ 可开启玻璃帷幕 | 烟囱效应<br>热空气上升排出<br>CO2自然稀释 | 天井顶部设CO2传感器<br>自动开启排气扇(CO2>800ppm) |
| **穿堂风设计** | L1-4F完全开放<br>无密闭墙面 | 横向气流贯穿<br>风速>0.5m/s | - |
| **绿化缓冲带** | 屋顶公园1公顷<br>垂直绿墙 | 植物吸收污染物<br>提升空气质量 | 每季检测PM2.5、VOCs |
| **高楼层配置** | 住宅位于11F以上 | 远离地面污染源<br>空气质量优 | - |

**实测数据**（WOHA Architects提供）：
- 自然通风可达室内时间**60%**（新加坡标准建筑仅30%）
- 人工照明需求减少**40%**
- CO2平均浓度**650ppm**（优于800ppm标准）

### 4.2 机械通风系统

```
┌─────────────────────────────────────────┐
│ 住宅区(11-21F)  : 分户空调(自行控制)   │
```

### 4.3 分流动线（Segregated Circulation）

#### **垂直分流**

```
专用电梯系统：
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│ 医疗电梯x2  │  │ 住宅电梯x2  │  │ 托婴电梯x1  │

  医疗患者         长者居民          幼儿家长

```

- 每4小时酒精擦拭→改为**每小时**

### 4.5 健康筛检 SOP

#### **新竹推估（赤土崎简化版）**

**假设条件**：
- ❌ 不建长者住宅（成本太高）

### 5.4 创新收入模式（新竹独有优势）

#### **竹科企业托婴名额认购制**

**模式**：
- 企业预购**年度托婴名额**（如10个名额/年）
- 费用：NT$ 18万/名额/年（市价75%折扣）
- 企业员工优先使用，空缺时开放候补

**财务试算**（30个企业名额）：
- 30名额 × NT$ 18万 = **NT$ 540万/年**
- 涵盖营运缺口50%

**企业效益**：
- ESG报告书加分（员工福祉）
- 竹科离职率↓（托婴在工作地附近）
- 税务抵减（公益捐赠）

### 7.4 新竹独有优势（🌟 Unique Strengths）

**可超越新加坡的地方**：

1. **竹科企业CSR**
   - 新加坡：政府主导，企业参与少
   - 新竹：**台积电、联发科、联电**等企业ESG需求强
   → 可建立"**企业认养制**"（名额预购+活动赞助）

2. **24H临托需求**
   - 新加坡：标准工时，无夜间需求
   - 新竹：**竹科轮班制、夜班工程师**需求高
   → 可加入"**夜间临托**"（参考日本一時保育）

3. **数字化程度**
   - 新加坡：SafeEntry等政府系统完善
   - 新竹：可整合**新竹市民卡**、**健保快易通**
   → 打造"**超级App**"（预约托婴+长照+健康站+活动报名）

4. **在地文化**
   - 新加坡：多元族群（华/马来/印度语）
   - 新竹：可强化**客家文化传承**（老人教客语、擂茶、花布）
   → 代际活动具在地特色
