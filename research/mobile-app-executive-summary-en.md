## UI/UX Best Practices for Elderly Users

### Visual Design
- **Font size**: Minimum 16pt for body text, larger for critical info
- **Touch targets**: 44x44 pixels minimum (Apple HIG), 48x48+ recommended
- **High contrast**: Provide high-contrast mode option
- **Simple colors**: Avoid blue/purple and yellow/green combinations

### Navigation
- **Simplified paths**: Minimize sublevels, single-function menus
- **Always visible**: "Back" and "Home" buttons always accessible
- **Favor taps over gestures**: Precision gestures difficult for elderly

### Support Features
- **Contextualized help**: Step-by-step instructions
- **Voice controls**: Siri/Google Assistant integration
- **Text-to-speech**: For users with reading difficulties
- **No disappearing messages**: Let users close them manually

### Design Process
- **Participatory design**: Co-design with actual elderly users
- **User testing**: Test with diverse age groups (60-65, 65-70, 70-75+)
- **Iterative feedback**: Continuous improvement based on user input
